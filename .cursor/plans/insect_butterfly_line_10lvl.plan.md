# 蝴蝶 / 昆虫线：替换茶饮 + 10 级产品 + 工具链设计

## 决策摘要

- **主题**：**不做卵阶**；**L1 毛毛虫 → L2 蛹 → L3 起全部为成蝶**；**L3–L10 各对应一种真实蝴蝶的中文俗名**（禁止「双蝶起舞」等状态式命名）；成蝶 **禁止黑/深灰褐作主色**，须 **鲜艳、宝石色与 pastel 向艺术加工**，靠 **斑纹结构** 认种；整体 **Q 版、无写实恐虫**（避免蜈蚣、蟑螂等）。
- **未上线**：可 **删除 `DrinkLine.TEA` / `tool_tea_set` / `drink_tea_*`**，改用新枚举与新 ID，无需兼容旧档。
- **产品**：**10 级**（仅此一条饮品位产品线升到 10；冷饮 / 甜品仍可维持 8 级，或后续再统一，本方案不强制改二线）。
- **工具**：维持 **5 级工具链**（1–2 级仅合成，**3 级起可产出**）。**五档全部是「捕虫网」**，靠 **框形（圆/椭/方）、网眼疏密、圈与柄材质（木/金属/镶边）、体积感** 拉开等级；**不出现花盆、土、植物、花、叶、枝、温室棚** 等，与种植/花艺工具彻底分家。命名：**小网 / 虫网 / 抄网 / 密网 / 闪网**。

## 工程侧要点（10 级与产出表）

- 现有「茶饮/烘焙」共用 [`SHARED_PRODUCE_TABLE_LV3`](src/config/BuildingConfig.ts) 最高只掷到 **产品 L7**（因饮品原为 8 级）。昆虫线 **必须** 单独定义 `INSECT_PRODUCE_TABLE_L3`～`L5`（命名示例），使 **T5 工具可掷出 1～10 级**（高等级低权重）。
- [`drinkDeliverHuayuanForLevel`](src/config/OrderHuayuanConfig.ts) 对任意 `level` 有解析式，**L9–L10 会自动有单价**，无需改公式；若希望昆虫线比冷饮更「稀有链」可再设 `insectDeliverHuayuanForLevel`（可选）。
- **品类命名**：推荐新增 **`Category.POLLINATOR`（或 `GARDEN_INSECT`）** + **`InsectLine.BUTTERFLY`**（或 `METAMORPHOSIS`），与 `Category.DRINK` 并列，订单与图鉴分支更清晰。备选是继续挂在 `Category.DRINK` 下仅加 `DrinkLine.INSECT`（改得快但语义别扭）。

## 世界观一句话

野采与观察用的 **捕虫网一路升级**（更趁手、更专业、更「闪亮」），点按产出毛毛虫 / 蛹 / 成蝶各阶段；**画面里只出现网具与昆虫**，不把花店种植道具画进图标（童话感，不要恐怖写实虫）。

## 工具链（5 级）— 替换原「茶具」（纯网具阶梯）

**器物语言唯一**：全是 **手持捕虫网**，不出现罩、笼、棚、盆、植物配景。与园艺线 **铲/壶/育苗/温室** 剪影完全错开。

**分级逻辑**：**柄长与环径变大** → **框形变化（圆→椭→方/扁）** → **网眼加密** → **圈边装饰与网面微闪**（仍是一张网，不要变成别类道具）。

| 等级 | 暂定名 | 定位 | 可产出 | 画图要点（一句话） |
|------|--------|------|--------|---------------------|
| **T1** | **小网** | 仅合成 | 否 | **短木柄 + 小圆环**，软白 **疏网眼**，整体迷你。 |
| **T2** | **虫网** | 仅合成 | 否 | **长直柄 + 大圆环**，白网略 **挺括**（尼龙感），与 T1 差在 **明显更长、环更大**。 |
| **T3** | **抄网** | **可产出** | 是 | **椭圆环 + 深袋网兜**，网眼比 T2 **略细**；剪影从「圆饼」变 **长椭圆+下垂袋形**。 |
| **T4** | **密网** | 可产出 | 是 | **圆环或微方圆环**，**深色木/金属圈加厚**，**密眼网**；专业感，与浅色系 T2 拉开。 |
| **T5** | **闪网** | 可产出 | 是 | 在 **密网** 基础上 **细金/银镶边** 或 **双环叠套**，网面可 **极淡珠光/闪点**（仍是织物网，不要变金属兜）；掷 **L2–L10**，**L8+ 权重明显低**。 |

**产出档（建议比例，落地时整数化并微调平衡）**

- **INSECT_T3**：`[[1, 88], [2, 12]]`
- **INSECT_T4**：`[[1, 35], [2, 35], [3, 22], [4, 8]]`
- **INSECT_T5**：`[[2, 8], [3, 18], [4, 22], [5, 22], [6, 15], [7, 10], [8, 3], [9, 1], [10, 1]]`  
  - 若希望终局更肝：可把 9、10 再减半，靠合成为主。

（冷饮线 `tool_mixer` 仍用现有 `MIXER_PRODUCE_TABLE_*`，与昆虫表 **独立**。）

## 产品链（10 级）— L1 虫 · L2 蛹 · L3–L10 八种名蝶

**结构**：**L1** 单阶毛毛虫；**L2** 单阶蛹；**L3–L10** 共 **8 级成蝶**，每一级 **物品名 = 一种蝴蝶的中文通用名**（可再加括注学名供美术检索）。**不出现「双蝶」「起舞」「灵翼」等非物种名。** **不含卵。**

**选种思路**（可查百科/图鉴核对翅面）：兼顾 **常见 → 华丽 → 高辨识度斑纹 → 名种**；**小图以斑纹结构认种**，**色相允许游戏向夸张**，须符合下节「成蝶配色总则」。

**构图硬规则**：**不要花、不要叶、不要枝、不要苔、不要盆**；背景 **纯色 / 柔渐变 / 淡光斑**；**每只图标仅一只蝶**，标准展翅或侧飞，突出 **该物种典型斑纹结构**（Q 版简化即可）。

**幼虫 / 蛹**：毛毛虫 **嫩绿、黄绿、奶黄** 等鲜亮软色；蛹 **金绿、蜜黄、薄荷珠光**（**悬浮**，无丝无挂点）。

### 成蝶配色总则（强制）

- **禁止以黑色、深灰、褐灰作为翅面主色**（主色指占翅面积 **约四成以上** 的底色）；允许 **细线、描边、小斑** 用深色勾形。
- **整体走「宝石色 + pastel + 高饱和点缀」**：青绿、湖蓝、品红、柠黄、蜜金、薰衣草、珊瑚橙、电光紫等；可 **渐变、珠光、闪粉**（merge 休闲向），与棋盘 **冷饮冰蓝、甜品粉橙** 仍靠 **蝶形与鳞翅斑纹结构** 区分。
- **认种靠「花纹结构」不靠「写实深色」**：如玉带凤蝶保留 **中央白带形态**，底色可改为 **宝蓝、深青或玫紫**；猫头鹰环蝶保留 **大眼斑构图**，底可改为 **莓粉、薄荷或淡紫渐变**。

| 等级 | 暂定名（物种） | 学名（供检索，实现时可不写进 UI） | 视觉锚点（一句话，鲜艳向艺术加工） |
|------|----------------|-----------------------------------|----------------------------------------|
| **L1** | **小毛毛虫** | — | Q 短胖虫，**黄绿/奶黄** 体节，侧视。 |
| **L2** | **金茧蛹** | — | **金绿 + 蜜黄高光** 蛹壳，悬浮居中。 |
| **L3** | **菜粉蝶** | *Pieris rapae* | **奶白至淡薰衣草翅**，翅尖用 **薄荷绿或淡橙小斑** 点题，**避免大块黑**。 |
| **L4** | **柑橘凤蝶** | *Papilio xuthus* | **柠黄 / 芽绿** 主色，纵带与凤尾用 **青绿描边 + 少量深青线**；整体明亮。 |
| **L5** | **玉带凤蝶** | *Papilio polytes* | **保留中央白带形**；底色改 **电光蓝、宝蓝或品红底**（择一），深色仅勾边。 |
| **L6** | **青凤蝶**（**翠凤蝶**） | *Graphium sarpedon* | **天青、湖蓝、薄荷** 占翅面大部分，横带 **加亮**，深色只做 **细脊线**。 |
| **L7** | **金裳凤蝶** | *Troides aeacus* | **蜜金、杏黄、浅橙闪** 为主；外缘 **珊瑚粉或浅褐细边**，**不要大面积乌褐底**。 |
| **L8** | **猫头鹰环蝶** | *Caligo eurilochus* 等 | **大眼斑结构保留**，斑心 **亮橙、电黄**；底色 **莓粉渐变、淡紫或蒂芙尼蓝**，**禁止写实灰褐大块面**。 |
| **L9** | **大蓝闪蝶** | *Morpho menelaus* | **高饱和青蓝 → 钴紫渐变** 金属闪，**浅青高光边**，翅形宽圆。 |
| **L10** | **金斑喙凤蝶** | *Teinopalpus aureus* | **翠绿、草绿** 为主，**金黄斑块** 提亮；金属线 **香槟金**，**避免墨绿发黑**。 |

**备选替换**（偏艳、易衔接）：**红锯蛱蝶**（*Cethosia biblis*，橙红几何边）；**玫瑰绡眼蝶**（*Cithaerias aurorina*，粉透翅）；**光明女神闪蝶**（*Morpho rhetenor*，紫蓝幻彩，与大蓝闪蝶二选一防撞色）；**大紫蛱蝶**（*Sasakia charonda*，宜 **紫红 + 粉白斑**，忌黑褐主底）。替换时仍须 **单蝶 + 物种专名**。

**文案注意**：鲜花线有 **「蝴蝶兰」**；本链图鉴写清 **物种蝴蝶**，与植物区分。

## 与「纯昆虫线」的取舍

- **本链锁死为鳞翅目成蝶物种名**；若以后要甲虫线应另开链，不与毛毛虫–蛹衔接。
- **图标层零植物元素**（花叶枝苔盆皆无），与鲜花/绿植/包装线在画面上 **硬隔离**。

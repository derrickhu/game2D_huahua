《花语小筑》Phaser 3 开发设计方案

一、技术选型总览

1. 核心技术栈

• 游戏引擎：Phaser 3.80+（最新稳定版）
• 开发语言：TypeScript 5.x（强类型，团队协作必备）
• 构建工具：Vite 5.x（极速 HMR，Tree Shaking，微信小游戏打包友好）
• UI 插件：rexrainbow/phaser3-rex-plugins（ScrollablePanel、GridSizer、Dialog 等）
• 图集工具：TexturePacker / Free Texture Packer（图集合并，减少 DrawCall）
• 代码规范：ESLint + Prettier
• 版本管理：Git

2. 目标平台

• 主平台：微信小游戏
• 适配方案：weapp-adapter（桥接 Canvas/Audio/Touch/Storage API）
• 设计分辨率：750×1334（竖屏，iPhone 6/7/8 基准）
• 缩放模式：Phaser.Scale.FIT（等比缩放适配各种屏幕）
• 渲染模式：优先 WebGL，降级 Canvas

3. 运行环境约束

• 微信小游戏包体限制：主包 ≤ 4MB，分包 ≤ 2MB/个，总计 ≤ 20MB
• 远程 CDN 资源：大部分图集和音效走 CDN 按需下载
• 本地存储：wx.setStorageSync / wx.getStorageSync（存档数据）
• 网络通信：wx.request（排行榜、社交、活动数据）

2. Scene 切换逻辑

```typescript
// UIScene 中处理导航切换
// NavBar 点击事件 → 切换到目标 Scene

class UIScene extends Phaser.Scene {
  private currentSceneKey: string = 'MainScene';

  switchScene(targetKey: string): void {
    if (targetKey === this.currentSceneKey) return;

    // 休眠当前场景（保持状态，不销毁）
    this.scene.sleep(this.currentSceneKey);

    // 唤醒或启动目标场景
    if (this.scene.isSleeping(targetKey)) {
      this.scene.wake(targetKey);
    } else {
      this.scene.launch(targetKey);
    }

    // 将 UIScene 置于最顶层
    this.scene.bringToTop('UIScene');

    this.currentSceneKey = targetKey;
  }
}
```

关键设计原则：
• UIScene 始终运行在最顶层，所有页面共享顶部栏和底部导航栏
• 页面切换使用 sleep/wake 而非 stop/start，保持各 Scene 的内部状态
• MainScene 被 sleep 时，内部定时器（建筑CD/客人刷新/挂机产出）继续运行（Phaser TimerEvent 在 sleep 时暂停，需使用 this.time 的 paused 管理或在 wake 时重新计算差值）

```typescript
class MainScene {
  // 合成规则：同品类+同线+同等级+未满级
  // 关键改变：锁定物品(isReserved)仍可合成！
  // 新增：支持建筑材料合成和宝箱合成
  private canMerge(a: BoardItem, b: BoardItem): boolean {
    return a.category === b.category
      && a.line === b.line
      && a.level === b.level
      && a.level < a.maxLevel  // 花束<6，花饮<3
      && a !== b;

  }

  private canMergeChest(a: ChestItem, b: ChestItem): boolean {
    return ChestItem.canMergeChests(a, b);

  private doMerge(source: BoardItem, target: BoardItem, targetCell: Cell): void {
    // 如果被合成的物品是锁定的，通知客人系统解锁对应槽位
    if (source.isReserved) EventManager.emit(GameEvents.ITEM_CONSUMED, { item: source });
    if (target.isReserved) EventManager.emit(GameEvents.ITEM_CONSUMED, { item: target });

    // 销毁源和目标，放置新的高级物品
    const nextId = getNextLevelId(source.itemId);
    // ...合成特效+放置新物品

    // 通知棋盘格解锁管理器（合成可能触发相邻迷雾格解锁）
    const resultLevel = source.level + 1;
    this.boardUnlockManager.onMergeCompleted(targetCell, resultLevel);
  }

  // 拖拽释放时的目标判断（增强版，支持可窥视格跨格合成）
  private onDragDrop(draggedItem: BoardItem, targetCell: Cell): void {
    // 1. 如果目标是可窥视格，尝试跨格合成
    if (targetCell.isLocked() && targetCell.lockType === CellLockType.PEEK) {
      const sourceCell = this.board.getCellAt(draggedItem.row, draggedItem.col);
      this.boardUnlockManager.attemptPeekMerge(sourceCell, targetCell);
      return;
    }

    // 2. 如果目标是钥匙格，尝试金币解锁
    if (targetCell.isLocked() && targetCell.lockType === CellLockType.KEY) {
      this.boardUnlockManager.attemptKeyUnlock(targetCell);
      draggedItem.returnToOriginal();
      return;
    }

    // 3. 普通合成逻辑（物品/建筑材料/宝箱）
    const targetContent = targetCell.getContent();
    if (targetContent) {
      if (this.canMerge(draggedItem, targetContent as BoardItem)) {
        this.doMerge(draggedItem, targetContent as BoardItem, targetCell);
      }
      // 建筑材料合成、宝箱合成类似处理
    }
```

```typescript
    super(scene, 0, 0);
    this.config = config;

      this.remainingUses = config.maxUses;

    if (this.config.type === BuildingType.PERMANENT && this.cdRemaining > 0) {
      // CD中，提示并显示广告加速选项
      this.showCdAccelerateOption();
      return;
    }

      if (this.remainingUses <= 0) return;
      if (Date.now() - this.lastTapTime < (this.config as ConsumableBuildingConfig).tapIntervalMs) return;

    if (!EnergyManager.hasEnough(this.config.energyCost)) {

      return;
    }

    if (!this.hasAdjacentEmpty()) {
      Toast.show('棋盘已满，请先合成');
      return;
    }

    // 5. 初级建筑无选择，直接产出；中高级弹出选择浮层
    if (this.config.selectableLines.length <= 1) {
      const line = this.config.selectableLines[0] || this.getDefaultLine();
      this.produce(line);
    } else {

  produce(line: string): void {
    // 扣除体力
    EnergyManager.consume(this.config.energyCost);

    // 计算产出
    const outputCount = Phaser.Math.Between(
      this.config.outputCount[0],
      this.config.outputCount[1]
    );
    const items: string[] = [];
    for (let i = 0; i < outputCount; i++) {
      const level = rollOutputLevel(this.config.outputTable);
      items.push(`${line}_${level}`);
    }

    // 永久型：启动CD
    if (this.config.type === BuildingType.PERMANENT) {
      this.startCooldown((this.config as PermanentBuildingConfig).cooldownMs);
    }

    // 消耗型：扣减使用次数
    if (this.config.type === BuildingType.CONSUMABLE) {
      this.remainingUses--;
      this.lastTapTime = Date.now();
      this.updateUsesDisplay();

      // 用尽则消失
      if (this.remainingUses <= 0) {
        this.onExhausted();
      }
    }

    // 发送产出事件
    EventManager.emit(GameEvents.BUILDING_PRODUCED, {
      buildingId: this.buildingId,
      items,
      row: this.row,
      col: this.col,
    });
  }
```

```typescript
  // 花束建筑合成线（4级）

      level: 4, name: '花房设计图', convertsToBuilding: 'building_flower_3' },
    // 两个 flower_bp_4 合成 → 自动转化为 building_flower_4（星光花房）
  ];

  // 饮品建筑合成线（3级）
  private static DRINK_CRAFT_CHAIN: BuildingMaterialConfig[] = [
    { id: 'drink_bp_1', materialCategory: BuildingMaterialCategory.DRINK_BLUEPRINT,
      level: 1, name: '饮品蓝图碎片' },
    { id: 'drink_bp_2', materialCategory: BuildingMaterialCategory.DRINK_BLUEPRINT,
      level: 2, name: '调饮器具箱', convertsToBuilding: 'building_drink_1' },
    { id: 'drink_bp_3', materialCategory: BuildingMaterialCategory.DRINK_BLUEPRINT,
      level: 3, name: '吧台设计图', convertsToBuilding: 'building_drink_2' },
    // 两个 drink_bp_3 合成 → 自动转化为 building_drink_3（花饮工坊）
  ];

  // 判断两个建筑材料是否可以合成
  canMergeMaterial(a: string, b: string): boolean {
    return a === b && this.getNextLevel(a) !== null;
  }

  // 合成建筑材料
  mergeMaterial(materialId: string): { resultId: string; autoBuilding?: BuildingConfig } {
    const chain = this.getChain(materialId);
    const current = chain.find(m => m.id === materialId);
    if (!current) throw new Error(`Unknown material: ${materialId}`);

    const nextLevel = current.level + 1;
    const next = chain.find(m => m.level === nextLevel);

    if (next) {
      // 合成为下一级材料
      const result: any = { resultId: next.id };
      // 如果下一级有自动转化建筑，则同时返回建筑配置
      if (next.convertsToBuilding) {
        result.autoBuilding = BuildingData.getConfig(next.convertsToBuilding);
      }
      return result;
    } else {
      // 已是最高级材料，两个最高级合成 = 顶级建筑
      const topBuildingId = this.getTopBuilding(materialId);
      return { resultId: topBuildingId, autoBuilding: BuildingData.getConfig(topBuildingId) };
    }
  }

  // 建筑材料合成后的处理（在 MergeManager 中调用）
  onMaterialMerged(resultId: string, cell: Cell): void {
    const mergeResult = this.mergeMaterial(resultId);

    if (mergeResult.autoBuilding) {
      // 自动转化为建筑：在当前格子放置新建筑
      const building = new Building(cell.scene, mergeResult.autoBuilding);
      building.row = cell.row;
      building.col = cell.col;
      cell.placeBuilding(building);

      // 播放建筑生成特效
      BuildingEffect.playBuildingCreated(cell.scene, cell.x, cell.y);
      AudioManager.playSfx('sfx_building_craft');

      // 图鉴记录
      EventManager.emit(GameEvents.BUILDING_UNLOCKED, { buildingId: mergeResult.autoBuilding.id });

      // 花愿奖励
      const wishReward = mergeResult.autoBuilding.type === BuildingType.PERMANENT ? 10 : 0;
      if (wishReward > 0) CurrencyManager.addWish(wishReward);

      Toast.show(`合成成功！获得 ${mergeResult.autoBuilding.name}`);
    } else {
      // 只是升级材料，放回格子
      cell.placeMaterial(mergeResult.resultId);
    }
  }

  private getChain(materialId: string): BuildingMaterialConfig[] {
    if (materialId.startsWith('flower_bp')) return BuildingCraftManager.FLOWER_CRAFT_CHAIN;
    if (materialId.startsWith('drink_bp')) return BuildingCraftManager.DRINK_CRAFT_CHAIN;
    return [];
  }

  private getNextLevel(materialId: string): string | null {
    const chain = this.getChain(materialId);
    const current = chain.find(m => m.id === materialId);
    if (!current) return null;
    const next = chain.find(m => m.level === current.level + 1);
    return next?.id || null;
  }

  private getTopBuilding(materialId: string): string {
    if (materialId.startsWith('flower_bp')) return 'building_flower_4'; // 星光花房
    return 'building_drink_3'; // 花饮工坊
  }
}
```

```typescript
class BoardUnlockManager {
  private board: Board;

  constructor(board: Board) {
    this.board = board;
  }

  // ===== 核心解锁机制 =====

  // 1. 相邻合成解锁（核心解锁方式）
  // 当玩家在已解锁格子中成功合成新物品时调用
  onMergeCompleted(cell: Cell, resultLevel: number): void {
    // 计算波及解锁数量
    let unlockCount = 1;
    if (resultLevel >= 3) unlockCount = Phaser.Math.Between(1, 2);
    if (resultLevel >= 5) unlockCount = 2;

    // 获取相邻的迷雾格（上下左右4方向）
    const adjacentFogCells = this.getAdjacentCellsByType(cell, CellLockType.FOG);

    // 按优先级排序后解锁
    adjacentFogCells.sort((a, b) => (a.unlockPriority || 0) - (b.unlockPriority || 0));
    const toUnlock = adjacentFogCells.slice(0, unlockCount);

    for (const fogCell of toUnlock) {
      this.unlockCell(fogCell, 'merge_ripple');
    }
  }

  // 2. 跨格合成解锁（可窥视格专属）
  // 当玩家将物品拖到可窥视格上时调用
  attemptPeekMerge(sourceCell: Cell, peekCell: Cell): boolean {
    if (peekCell.lockType !== CellLockType.PEEK) return false;

    const sourceItem = sourceCell.getItem();
    const peekItem = peekCell.getHiddenItem();
    if (!sourceItem || !peekItem) return false;

    // 检查合成条件
    if (sourceItem.category !== peekItem.category
      || sourceItem.line !== peekItem.line
      || sourceItem.level !== peekItem.level
      || sourceItem.level >= sourceItem.maxLevel) {
      return false;
    }

    // 执行跨格合成
    const nextId = getNextLevelId(sourceItem.itemId);

    // 消耗源物品
    sourceCell.removeItem();

    // 解锁可窥视格（变为空格）
    this.unlockCell(peekCell, 'peek_merge');

    // 合成结果放入源格子（或最近空格）
    const resultCell = sourceCell.isEmpty() ? sourceCell : this.board.findNearestEmpty(sourceCell);
    if (resultCell) {
      const newItem = BoardItem.create(sourceCell.scene, nextId);
      resultCell.placeItem(newItem);
      MergeEffect.play(resultCell.scene, resultCell.x, resultCell.y);
    }

    // 波及解锁周围迷雾格
    this.onMergeCompleted(peekCell, sourceItem.level + 1);

    return true;
  }

  // 3. 钥匙格解锁（金币解锁）
  attemptKeyUnlock(keyCell: Cell): boolean {
    if (keyCell.lockType !== CellLockType.KEY) return false;
    const price = keyCell.lockKeyPrice || 0;

    if (!CurrencyManager.hasEnoughGold(price)) {
      Toast.show('金币不足');
      return false;
    }

    Dialog.show({
      title: '解锁格子',
      content: `消耗 ${price} 金币解锁此格？`,
      confirmText: '解锁',
      onConfirm: () => {
        CurrencyManager.spendGold(price);
        this.unlockCell(keyCell, 'key_unlock');
      },
    });
    return true;
  }

  // ===== 通用解锁逻辑 =====

  private unlockCell(cell: Cell, animationType: string): void {
    const hiddenContent = cell.hiddenContent;

    // 更新格子状态
    cell.unlock();

    // 播放解锁动画
    switch (animationType) {
      case 'merge_ripple':
        UnlockEffect.playFogClear(cell.scene, cell.x, cell.y);
        break;
      case 'peek_merge':
        UnlockEffect.playPeekMerge(cell.scene, cell.x, cell.y);
        break;
      case 'key_unlock':
        UnlockEffect.playKeyUnlock(cell.scene, cell.x, cell.y);
        break;
    }

    // 放置隐藏内容
    if (hiddenContent && hiddenContent !== 'empty') {
      this.scene.time.delayedCall(400, () => {
        this.placeHiddenContent(cell, hiddenContent);
      });
    }

    // 检查连锁提示
    this.checkChainHints(cell);

    // 发送解锁事件
    EventManager.emit(GameEvents.CELL_UNLOCKED, {
      row: cell.row,
      col: cell.col,
      animationType,
    });
  }

  // 放置隐藏物品到解锁的格子中
  private placeHiddenContent(cell: Cell, contentId: string): void {
    if (contentId.startsWith('item:')) {
      // 花束/花饮物品
      const itemId = contentId.replace('item:', '');
      const item = BoardItem.create(cell.scene, itemId);
      cell.placeItem(item);
    } else if (contentId.startsWith('material:')) {
      // 建筑材料
      const materialId = contentId.replace('material:', '');
      cell.placeMaterial(materialId);
    } else if (contentId.startsWith('building:')) {
      // 消耗型建筑
      const buildingId = contentId.replace('building:', '');
      const config = BuildingData.getConfig(buildingId);
      if (config) {
        const building = new Building(cell.scene, config);
        cell.placeBuilding(building);
      }
    } else if (contentId.startsWith('chest:')) {
      // 宝箱
      const chestId = contentId.replace('chest:', '');
      const config = ChestData.getConfig(chestId);
      if (config) {
        const chest = new ChestItem(cell.scene, config, cell.row, cell.col);
        cell.placeChest(chest);
      }
    } else if (contentId.startsWith('gold:')) {
      // 金币奖励（直接加到货币）
      const amount = parseInt(contentId.replace('gold:', ''));
      CurrencyManager.addGold(amount);
      CurrencyFlyEffect.play(cell.scene, cell.x, cell.y, 'gold', amount);
    }

    // 物品揭示动画（弹性放大）
    UnlockEffect.playItemReveal(cell.scene, cell.x, cell.y);
  }

  // 连锁提示：解锁后检查新内容是否可与周围物品合成
  private checkChainHints(cell: Cell): void {
    const item = cell.getItem();
    if (!item) return;

    const neighbors = this.board.getAdjacentCells(cell);
    for (const neighbor of neighbors) {
      if (!neighbor.isLocked() && neighbor.hasItem()) {
        const neighborItem = neighbor.getItem();
        if (neighborItem && this.board.canMerge(item, neighborItem)) {
          // 高亮提示可合成
          item.showMergeHint();
          neighborItem.showMergeHint();
        }
      }
    }
  }

  // ===== 拖拽交互增强：支持拖到可窥视格 =====

  // 在 MainScene.setupDragAndDrop() 中的 drop 判断增加：
  // if (targetCell.lockType === CellLockType.PEEK) {
  //   boardUnlockManager.attemptPeekMerge(sourceCell, targetCell);
  // }

  // ===== 可窥视格的视觉提示 =====

  // 当玩家拖拽一个物品时，遍历所有相邻可窥视格
  // 如果可窥视格中的物品与被拖拽物品可合成，则该格子边框闪烁金色
  onDragStart(draggedItem: BoardItem): void {
    const dragCell = this.board.getCellAt(draggedItem.row, draggedItem.col);
    const neighbors = this.board.getAdjacentCells(dragCell);

    for (const cell of neighbors) {
      if (cell.lockType === CellLockType.PEEK) {
        const peekItem = cell.getHiddenItem();
        if (peekItem
          && peekItem.category === draggedItem.category
          && peekItem.line === draggedItem.line
          && peekItem.level === draggedItem.level) {
          cell.showPeekMergeHint(true);
        }
      }
    }
  }

  onDragEnd(): void {
    // 清除所有可窥视格的高亮提示
    this.board.getAllCells()
      .filter(c => c.lockType === CellLockType.PEEK)
      .forEach(c => c.showPeekMergeHint(false));
  }

  // ===== 辅助方法 =====

  private getAdjacentCellsByType(cell: Cell, lockType: CellLockType): Cell[] {
    return this.board.getAdjacentCells(cell)
      .filter(c => c.isLocked() && c.lockType === lockType);
  }

  // 获取当前已解锁格子数（用于进度统计）
  getUnlockedCount(): number {
    return this.board.getAllCells().filter(c => !c.isLocked()).length;
  }

  getTotalCount(): number {
    return this.board.getAllCells().length;
  }
}
```

```typescript
class Cell extends Phaser.GameObjects.Container {

  // 初始化为锁定格
  initAsLocked(preset: CellPreset): void {
    this.locked = true;
    this.lockType = preset.initState as CellLockType;
    this.hiddenContent = preset.presetContent;
    this.lockKeyPrice = preset.keyPrice;
    this.unlockPriority = preset.unlockPriority;

    switch (this.lockType) {
      case CellLockType.FOG:
        this.drawFogOverlay();
        break;
      case CellLockType.PEEK:
        this.drawPeekOverlay();
        this.drawHiddenItemPreview();
        break;
      case CellLockType.KEY:
        this.drawKeyOverlay();
        break;
    }
  }

  isLocked(): boolean { return this.locked; }

  unlock(): void {
    this.locked = false;
    this.lockType = undefined;
    this.fogOverlay?.destroy();
    this.peekOverlay?.destroy();
    this.lockIcon?.destroy();
    this.keyPriceText?.destroy();
    this.hiddenItemPreview?.destroy();
  }

  getHiddenItem(): { category: ItemCategory; line: string; level: number } | null {
    if (!this.hiddenContent || !this.hiddenContent.startsWith('item:')) return null;
    const itemId = this.hiddenContent.replace('item:', '');
```

```typescript
      const nextStep = GuideManager.STEPS[this.currentStep];
      if (nextStep.triggerCondition) {
        // 有触发条件，等待条件满足
        this.waitForCondition(nextStep.triggerCondition, () => {
          this.executeStep(nextStep);
        });
      } else {
        this.executeStep(nextStep);
      }
    } else {
      this.completeGuide();
    }
  }
```

八、性能优化策略

1. 渲染优化

• 图集合并：所有小图使用 TexturePacker 合成图集，单图集不超过 2048×2048
• 对象池：频繁创建/销毁的对象使用对象池（花朵特效粒子、货币飘字、客人）
• Scene.sleep：非活跃页面 sleep 而非 destroy，减少重建开销
• 可见性裁剪：棋盘外的花朵不渲染（setVisible(false)）
• 批量渲染：同图集元素集中绘制，减少 DrawCall

2. 内存优化

• 按需加载：换装高清资源、装修高级装饰、活动资源走 CDN 懒加载
• 及时释放：切换 Scene 时释放非共享的大图纹理
• 图集复用：尽量多个元素共享同一图集
• 帧动画优化：客人走路等简单动画使用3~6帧，不追求高帧数

3. 逻辑优化

• 定时器合并：相同间隔的定时器合并处理（如多个永久型建筑的 CD 使用 BuildingManager 的全局1秒 timer 统一更新，体力恢复使用 EnergyManager 的独立3分钟 timer）
• 事件节流：拖拽过程中的碰撞检测每 3 帧检测一次，而非每帧
• 离线计算：离线收益在登录时一次性计算，不模拟离线过程
• 延迟初始化：图鉴/社交等二级页面首次进入时才初始化

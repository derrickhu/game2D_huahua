===== 全局美术风格参考 =====

统一风格提示词前缀（所有棋盘物品通用）：
"hand-painted 2D game icon with soft 3D shading, slightly angled natural pose with gentle dynamic curve, single unified light source from top-left, smooth gradient shading and soft volume highlights on every surface to create pseudo-3D depth, absolutely NO interior linework or outlines between parts, only a very thin 1-pixel outer contour outline that matches each local surface color slightly darker - white petals get a light warm gray outline and green stems get a darker green outline - each part's outline color follows its own local color not the overall object color, clean simplified shapes with minimal interior detail, bold readable silhouette even at 80x80px, soft pastel color palette, solid flat-color background #F5F0EB for easy background removal, overall composition centered on canvas with object filling at least 90 percent of the canvas, lively but balanced layout, PNG game asset, no drop shadow, no cast shadow, no shadow on ground"

简称：[BASE_STYLE]

风格要点说明：
• 核心风格：2D手绘 + 仿3D体积感（柔和高光体现立体感），非纯扁平、非真3D渲染
• 视角与姿态：允许物品有自然微倾斜、弧度和动态感（花茎微弯、叶子不对称等），但整体重心居中，画面不偏移
• 光源统一：所有物品左上方打光，花瓣/物体表面有柔和高光，但**不带底部投影/阴影**（阴影由游戏引擎运行时添加）
• 描边风格：**仅最外轮廓**有极细（1px级）的描边，且描边颜色**跟随局部表面色**——白色花瓣外缘用浅暖灰/米白色描边、绿色茎叶用深绿描边、黄色花心用深黄描边，每个部件的描边色取该部件自身颜色的略深色，**不是统一用绿色或其他单一颜色描所有部分**。物品内部**完全不画线条**，花瓣之间、叶子与茎之间等内部结构完全靠渐变色和高光来区分体积，不用线条分割，营造柔和自然的伪3D体积感
• 造型简洁：用概括的大色块表现物品，避免复杂纹理和碎细节。高级物品通过「包装升级+体积增大+材质光泽提升」体现等级差异，不靠堆叠花朵数量或装饰细节
• 轮廓清晰：缩到80×80px时仍能一眼辨认品类和等级
• 背景统一：所有物品使用统一浅暖灰底色 #F5F0EB 出图，便于批量抠图去底（避免AI生成随机背景色）
• 色调辨识：每个花系/饮品线保持鲜明主色调区分，格子中一眼可分辨
• 构图要求：物品在画布中**完全居中**，占满画布至少90%，只留极窄边距。抠图后直接缩放到格子尺寸使用，不再二次裁切，因此生成时就要按格子内最终表现来构图
• 出图尺寸：统一 512×512px 出图，游戏内按需缩放

十二、关键注意事项
===================================================================

1. 所有图片必须 PNG 透明背景，不能有白底
2. 斜45°俯视的物品（建筑、花店场景）需要统一光源方向（左上方来光）
3. 花束三个花系的色调区分必须明显（暖黄绿 / 粉紫红 / 金蓝银），花饮三条饮品线的色调和造型也需明确区分（茶饮暖茶棕 / 冷饮冰蓝 / 甜品粉橙）
4. Q版角色头身比统一2.5~3头身，脸部风格一致
5. 换装立绘头身比统一5~6头身，与Q版是同一角色的不同比例
6. 所有图标/按钮保持统一圆角弧度和阴影风格
7. 图片命名严格按照编号规范，便于程序引用
8. 如使用AI生成，需人工二次调整确保风格统一性
9. 图集合并时注意单图集不超过2048×2048，留足边距避免边缘渗色

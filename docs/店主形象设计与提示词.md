# 店主整体形象设计方案

## 设计原则

- **风格统一**：手绘2D + 柔和3D体积感，清新柔和、低饱和度、温馨明亮
- **角色一致性**：6套形象必须是同一个人，面部特征、肤色、体型完全相同
- **头身比**：4头身 Q版比例，可爱但能清晰展示服装细节
- **视角**：正面3/4侧身，自然站姿
- **背景**：统一浅暖米色 `#FFF8F0`
- **布局**：3列 × 2行网格，每格一套完整形象

## 角色基础特征（所有形象统一）

| 特征 | 描述 |
|------|------|
| 年龄 | 约18~20岁，年轻甜美 |
| 肤色 | 暖白偏粉 `#FFDDB8` |
| 眼睛 | 大而圆，温暖棕色，带高光 |
| 脸型 | 圆润鹅蛋脸，小巧可爱 |
| 表情 | 温暖微笑，带淡淡腮红 |
| 体型 | 娇小纤细 |

## 6套形象详细设计

### 1. 自然少女（默认形象）

> 清新自然的花店日常装扮，亲切温暖

| 下装 | 浅蓝色牛仔A字裙，及膝长度 |
| 鞋子 | 白色Mary Jane小皮鞋 |

| 色调 | 粉白+浅蓝，清新田园 |

| 裙装 | 白色+金色抹胸女神长裙，胸前精致花卉刺绣，裙摆层层叠叠渐变为花瓣造型，腰间金色藤蔓腰带 |

| 妆容 | 淡金色眼影、玫瑰花瓣唇色、微光腮红、颧骨轻微珠光 |
| 特效 | 人物周围有柔和的金色光晕和飘落的花瓣 |
| 色调 | 白+金+玫瑰+薰衣草紫，华丽神圣 |

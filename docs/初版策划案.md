1. 游戏名称：花花妙屋

2. 游戏类型：2D仿3D斜45°视角，合成经营+家园装修+店主换装微信小游戏

4. 目标用户：全年龄段女性休闲玩家，喜爱治愈、收集、养成、搭配、经营

5. 美术风格：清新柔和、低饱和度、温馨明亮，纯2D资源，伪3D透视

6. 核心优势：
• 保留四季物语全部成功玩法
• 新增换装养成线，女性吸引力更强
• 多花系短链合成，策略性更丰富

二、游戏核心五套玩法

1. 棋盘合成玩法（基础核心）—— 花束+花饮双品类合成

2. 客人组合订单经营 + 熟客养成（中期目标）

3. 花愿装修系统（长线家园收集）

4. 花露换装系统（独家时装收集）

5. 轻社交系统（微信生态传播）

三、玩法一：棋盘合成系统（花束+花饮双品类设计）

设计理念：现实中很多网红花店采用"鲜花+花饮"复合业态，我们将这一模式引入游戏。棋盘上同时存在花束和花饮两大品类，它们独立合成、互不干扰，但客人可以同时购买两种品类的商品，极大丰富了玩法层次。

1. 棋盘规则

• 主界面下方为固定合成棋盘，最大6×5=30格，初始仅开放中心4格，其余格子通过合成探索逐步解锁（详见"棋盘格渐进解锁系统"）

• 棋盘内包含：花束物品 + 花饮物品 + 花束建筑（永久型/消耗型）+ 饮品建筑（永久型/消耗型）+ 宝箱 + 建筑材料

• 支持拖拽合成、点击建筑产出（消耗体力）、点击宝箱（消耗体力）

• 花束建筑和饮品建筑共享棋盘空间，玩家需在建筑、物品、宝箱、建筑材料之间多方博弈

2. 拖拽合成（固定升级）<!--  -->

• 两个同品类同系同等级物品拖拽合并，100%合成高一级物品

• 无消耗、无失败、无冷却，合成带闪光特效与反馈

• 不同品类之间不可跨品类合成（花束不能和花饮合成）

• 同品类内不同系之间不可跨系合成

3. 品类一：花束系（3大花系，每系6级，共18种花束）

花系一：日常花系（基础产量高，满足初期订单）
1 小雏菊 → 2 向日葵 → 3 康乃馨 → 4 满天星花束 → 5 混搭花束 → 6 精致礼盒花

花系二：浪漫花系（中级品质，浪漫场景订单专用）
1 粉玫瑰 → 2 百合 → 3 郁金香 → 4 薰衣草花束 → 5 告白玫瑰礼盒 → 6 婚礼花艺

花系三：奢华花系（高级稀有，大单专用）
1 星空兰 → 2 生日花礼 → 3 星空花礼 → 4 鎏金花束 → 5 极光花礼 → 6 永恒花海典藏

合成规则：同花系N级 + N级 = N+1级
每条链仅需2^5=32个基础花即可合成顶级花，数值健康可控

4. 品类二：花饮系（3条饮品线，每线3级，共9种花饮）

花饮系为短链设计（仅3级），合成门槛低、产出节奏快，与花束系的长链形成节奏互补。

饮品线一：茶饮线（日常基础，产量高，简单订单专用）
1 花草茶 → 2 调味花茶 → 3 限定手作茶

饮品线二：冷饮线（中级品质，夏日/浪漫场景专用）
1 花果冰饮 → 2 花漾气泡水 → 3 梦幻花饮

饮品线三：甜品线（高级稀有，大单/组合订单专用）
1 花瓣饼干 → 2 花艺蛋糕 → 3 花宴甜品台

合成规则：同饮品线N级 + N级 = N+1级
每条链仅需2^2=4个基础饮品即可合成顶级饮品，快速成型

5. 双品类设计说明

• 花束系（6级长链）：合成周期长，单品价值高，是核心收入来源
• 花饮系（3级短链）：合成周期短，快速满足组合订单需求，节奏调味剂
• 两大品类视觉差异大：花束是花朵/花束/礼盒造型，花饮是杯子/蛋糕/甜品造型，玩家一目了然
• 花束和花饮不能跨品类合成，但客人可以同时需要花束+花饮，形成"生产两种品类→满足组合订单"的策略循环
• 总商品种类：花束18种 + 花饮9种 = 27种，收集驱动力翻倍
• 棋盘上花束建筑和饮品建筑共享有限格子，玩家需在"花束产能 vs 饮品产能 vs 合成空间"之间三方博弈，策略深度大幅提升

6. 建筑获取方式：合成获取（建筑合成线）

参考四季物语，建筑不再通过金币直接购买，而是通过独立的"建筑合成线"合成获得。建筑有专门的合成链路，初级建筑材料来源于宝箱奖励和棋盘锁定格中隐藏的建筑碎片。

（1）建筑合成线设计

花束建筑合成线（4级）：
1级 花束蓝图碎片 + 花束蓝图碎片 → 2级 花艺工具箱
2级 花艺工具箱 + 花艺工具箱 → 3级 花架组件
3级 花架组件 + 花架组件 → 4级 花房设计图

饮品建筑合成线（3级）：
1级 饮品蓝图碎片 + 饮品蓝图碎片 → 2级 调饮器具箱
2级 调饮器具箱 + 调饮器具箱 → 3级 吧台设计图

合成到顶级后，建筑自动生成在棋盘空格中（如无空格则提示腾出空间）。

（2）建筑材料来源

• 宝箱奖励：完成订单、每日任务、成就等固定获得宝箱，宝箱点击后产出建筑碎片（每个宝箱可点击3~5次，点完后消失）
• 锁定格隐藏：棋盘迷雾格/可窥视格中预置建筑碎片，解锁后获得
• 任务奖励：主线任务关键节点直接奖励中级建筑材料，加速首个建筑合成

（3）宝箱系统

宝箱是建筑碎片和额外资源的核心来源，属于消耗型物品：

• 宝箱等级：
  ◦ 铜宝箱（初级）：可点击3次，每次产出1个建筑碎片或少量金币
  ◦ 银宝箱（中级）：可点击5次，产出中级建筑材料或花愿/花露
  ◦ 金宝箱（高级）：可点击8次，产出高级建筑材料或稀有物品

• 宝箱也可以合成升级：2个铜宝箱合成1个银宝箱，2个银宝箱合成1个金宝箱

• 宝箱获取途径：
  ◦ 交付客人订单时固定掉落（每3~5单掉落1个铜宝箱）
  ◦ 每日任务完成奖励
  ◦ 成就达成奖励（银/金宝箱）
  ◦ 连续登录奖励
  ◦ 棋盘锁定格中预置

• 宝箱点击消耗体力（详见"体力系统"章节）
• 宝箱点完所有次数后自动消失，释放棋盘格子

7. 建筑两大类型：永久型（CD冷却）与 消耗型（用完消失）

建筑分为两种本质不同的类型，丰富策略层次：

（1）永久型建筑（Rechargeable Generator）

• 核心特征：点击产出物品后进入冷却期（CD），冷却结束后可再次点击，永不消失
• 每次点击消耗体力
• 冷却期间建筑显示倒计时，不可点击
• 玩家可选择观看广告立即重置CD（每日限3次）
• 永久型建筑是棋盘的核心产出引擎，需要通过建筑合成线合成获得

花束永久型建筑清单：

① 花艺操作台（初级花束建筑）
  • 产出：日常花系（70%1级、25%2级、5%3级）
  • 每次点击产出1个物品，消耗3体力
  • 冷却时间：30秒<!--  -->
  • 无需选择花系，固定产出日常花系
  • 获取：合成花束建筑合成线至2级（花艺工具箱）自动转化

② 包装台（中级花束建筑）
  • 装填选择：可选"日常花系"或"浪漫花系"
  • 产出概率：60%1~2级、30%3~4级、10%5级
  • 每次点击产出1个物品，消耗5体力
  • 冷却时间：2分钟
  • 获取：合成花束建筑合成线至3级（花架组件）自动转化

③ 小型温室（中高级花束建筑）
  • 装填选择：可选"浪漫花系"或"奢华花系"
  • 产出概率：55%1~2级、35%3~4级、10%5级
  • 每次点击产出1~2个物品，消耗8体力
  • 冷却时间：5分钟
  • 获取：合成花束建筑合成线至4级（花房设计图）自动转化

④ 星光花房（顶级花束建筑）
  • 装填选择：可选三个花系中的任意一个
  • 产出概率：40%3~4级、40%5级、20%6级
  • 每次点击产出1~2个物品，消耗12体力
  • 冷却时间：10分钟
  • 获取：消耗2个花房设计图（4级建筑材料）合成

饮品永久型建筑清单：

① 简易茶台（初级饮品建筑）
  • 产出：茶饮线（70%1级、25%2级、5%3级）
  • 每次点击产出1个物品，消耗3体力
  • 冷却时间：30秒
  • 无需选择饮品线，固定产出茶饮线
  • 获取：合成饮品建筑合成线至2级（调饮器具箱）自动转化

② 调饮吧台（中级饮品建筑）
  • 装填选择：可选"茶饮线"或"冷饮线"
  • 产出概率：55%1级、35%2级、10%3级
  • 每次点击产出1个物品，消耗5体力
  • 冷却时间：2分钟
  • 获取：合成饮品建筑合成线至3级（吧台设计图）自动转化

③ 花饮工坊（顶级饮品建筑）
  • 装填选择：可选三条饮品线中的任意一条
  • 产出概率：40%1级、35%2级、25%3级
  • 每次点击产出1~2个物品，消耗8体力
  • 冷却时间：5分钟
  • 获取：消耗2个吧台设计图（3级建筑材料）合成

（2）消耗型建筑（Exhaustible Generator）

• 核心特征：有固定的总产出次数，每次点击消耗1次使用次数+体力，次数用尽后建筑自动消失释放格子
• 不需要等待CD，只要有体力就能连续点击（每次点击间隔1秒防误触）
• 消耗型建筑通常比永久型建筑产出等级更高、概率更好，但总量有限
• 消耗型建筑来源更容易获得（宝箱直接产出、任务奖励、活动奖励等）
• 消耗型建筑鼓励玩家"趁有就用"，增加操作的紧迫感和爽快感

消耗型建筑清单：

① 花材礼盒（花束消耗型·初级）
  • 产出：随机花系1级花束
  • 总使用次数：5次，每次消耗2体力
  • 来源：铜宝箱、初级任务奖励

② 精选花篮（花束消耗型·中级）
  • 产出：指定花系1~3级花束（玩家可选花系）
  • 总使用次数：4次，每次消耗4体力
  • 来源：银宝箱、中级任务奖励

③ 花艺大师箱（花束消耗型·高级）
  • 产出：指定花系3~5级花束
  • 总使用次数：3次，每次消耗6体力
  • 来源：金宝箱、成就奖励、限时活动

④ 茶包盒（饮品消耗型·初级）
  • 产出：随机饮品线1级花饮
  • 总使用次数：5次，每次消耗2体力
  • 来源：铜宝箱、初级任务奖励

⑤ 调饮套装（饮品消耗型·中级）
  • 产出：指定饮品线1~2级花饮（玩家可选饮品线）
  • 总使用次数：4次，每次消耗4体力
  • 来源：银宝箱、中级任务奖励

⑥ 花饮臻选箱（饮品消耗型·高级）
  • 产出：指定饮品线2~3级花饮
  • 总使用次数：3次，每次消耗6体力
  • 来源：金宝箱、成就奖励、限时活动

（3）两种建筑的策略对比

| 维度 | 永久型建筑 | 消耗型建筑 |
|------|-----------|-----------|
| 获取难度 | 高（需合成建筑线） | 低（宝箱/任务直接获得） |
| 使用寿命 | 永久 | 用完消失（3~5次） |
| 产出节奏 | 慢（有CD冷却） | 快（无CD，连续点击） |
| 体力消耗 | 中等 | 较低 |
| 棋盘占用 | 永久占1格 | 临时占1格，用完释放 |
| 策略定位 | 长线稳定产出引擎 | 短期爆发、冲订单利器 |

8. 建筑设计说明

• 建筑通过合成获得而非金币购买，增加合成探索的深度和目标感（"我要凑齐碎片合出更好的建筑"）
• 永久型建筑是棋盘核心引擎，CD冷却控制产出节奏，避免无限刷物品导致的数值失控
• 消耗型建筑提供短期爆发手段，玩家在接到高难度组合订单时可以集中使用消耗型建筑快速凑齐物品
• 所有建筑点击均消耗体力，体力成为调控游戏节奏的核心资源（详见"体力系统"章节）
• 平衡性通过"建筑占棋盘格子"来控制——永久型建筑+消耗型建筑+物品+宝箱共享有限格子，玩家需在多者之间取舍
• 初级建筑无选择，降低新手门槛；中高级建筑增加产出线选择，根据订单需求主动规划
• 花束建筑和饮品建筑的获取节奏错开：前期通过主线任务加速获得首个花束建筑，10~15分钟后获得首个饮品建筑
• 饮品建筑只有3种（对比花束建筑4种），因为花饮系链路更短，不需要那么多层级的建筑
• 宝箱作为建筑碎片的核心来源，同时也是消耗型建筑的直接产出物，形成"做订单→得宝箱→开宝箱→得碎片/消耗型建筑→合成/使用→做更多订单"的正循环

四、玩法二：客人组合订单经营系统 + 熟客养成（核心赚钱循环）

1. 客人机制

• 客人随机在主界面刷新，头顶显示需求气泡（含1~3个需求槽位）

• 每个需求槽位显示一种商品图标（花束或花饮），带品类和花系标识

• 交付方式：自动锁定 + 手动确认交付（详见下方"交付机制"）

• 客人等级越高，需求种类越多、等级越高，奖励越丰厚

2. 交付机制：自动锁定 + 手动确认交付

核心规则：
• 当客人上门后，系统自动扫描棋盘上的物品，若存在满足某个需求槽位的物品，该物品会被自动锁定（显示锁定特效提醒+客人头像小标记）
• 锁定的物品仍然可以被拖动和合成——这是关键策略点：玩家可以选择把已锁定的低级物品拿去合成更高级的，腾出空间或满足更高级的订单
• 如果锁定物品被合成消耗，对应的需求槽位锁定状态自动取消，系统重新扫描棋盘寻找新的匹配物品
• 客人的多个需求槽位可以逐一被锁定——不需要一次性凑齐，棋盘上每新合成/产出一个匹配物品，就自动锁定到对应槽位
• 当客人的所有需求槽位都被锁定满足后，客人旁边出现"完成"按钮，玩家点击后触发交付动画
• 若棋盘上有多个同类物品可满足需求，优先锁定等级最低的（避免浪费高级物品）

交付流程：
① 客人上门 → 头顶显示需求气泡（1~3个需求槽位）
② 系统扫描棋盘 → 匹配物品自动锁定 → 物品显示锁定特效提醒
③ 未满足的槽位保持等待状态 → 玩家继续合成/产出 → 新物品自动锁定
④ 所有槽位满足 → 客人旁边出现"完成"按钮
⑤ 玩家点击"完成"按钮 → 锁定物品飞向客人 → 交付结算动画 → 获得金币+花愿/花露

策略深度说明：
• 锁定物品可以合成：玩家可以"抢走"已锁定给客人A的花去合成更高级的花，用于满足奖励更高的客人B，增加决策博弈
• 手动确认交付：全部满足后不会自动交付，玩家可以在点击"完成"前重新考虑——例如把锁定的低级花拿去合成后再等产出新的来替换，追求更优解
• 多客人竞争：当多个客人同时在场，锁定物品的取舍成为核心策略——先满足小单快速回金，还是攒物品满足大单拿高额奖励？
• 自动锁定省去了繁琐的拖拽操作，玩家只需专注于合成策略和交付时机的决策
• 锁定特效提醒清晰直观（光效+客人头像标记），玩家一目了然哪些物品已被预定

客人从"只买一种"升级为"组合购买"，不同客人类型有不同的需求数量和品类组合：

需求规则：
• 前期（未解锁饮品建筑时）：客人只需求花束类商品，需求槽位数为1
• 解锁饮品建筑后：客人开始出现花束+花饮的组合需求
• 客人需求槽位数随游戏进度增加（前期1个，中期1~2个，后期1~3个）
• 组合订单的总奖励高于同等难度的单品订单之和，激励玩家生产两种品类
• 客人等待时间随需求槽位数增加而延长，不会因为组合订单导致超时

4. 熟客养成系统（核心留存机制）

• 部分客人（学生少女、上班族、温柔妈妈、文艺青年、情侣、网红博主、贵妇、收藏家，共8位）为可养成熟客

• 每次交付成功，对应熟客好感度+1

• 好感度等级与奖励：

好感度Lv1（累计交付5次）：解锁客人小档案（姓名、性格、故事背景）
好感度Lv2（累计交付15次）：解锁专属订单（奖励翻倍）
好感度Lv3（累计交付30次）：解锁客人故事剧情（如"告白男生终于求婚成功"）
好感度Lv4（累计交付50次）：解锁专属头像框 + 大量花愿/花露奖励
好感度Lv5（累计交付80次）：客人外观升级 + 解锁限定图鉴

• 设计说明：熟客故事为短文本+静态插图，开发成本低但情感粘性极强，对女性用户留存效果显著

5. 订单体系

• 即时订单：客人上门，棋盘自动锁定匹配物品，凑齐后点击"完成"按钮交付

• 每日订单：固定3~5单，需求组合更复杂，奖励更高

• 限时高级订单：产出钻石+大量花愿

• 熟客专属订单：好感度Lv2后解锁，奖励翻倍，可能包含花束+花饮组合需求

6. 金币用途

• 解锁棋盘钥匙格（金币核心消耗出口）

• 购买体力回复药水（详见"体力系统"章节）

• 提升基础挂机产出速度

• 注：建筑不再通过金币购买，改为通过建筑合成线合成获得

7. 经营循环

合成花束/花饮 → 点击建筑产出（消耗体力，永久型等CD/消耗型用完消失）→ 自动锁定交付组合订单 → 赚金币+花愿/花露+宝箱 → 开宝箱得建筑碎片/消耗型建筑 → 合成更高级建筑 → 养熟客 → 更高组合订单

五、玩法三：花愿装修系统（对标四季物语星星家园，增加有限自由度）

1. 花愿获取途径

• 完成所有订单

• 交付客人

• 合成建筑（通过建筑合成线合成新建筑时获得）

• 图鉴收集

• 每日任务

• 成就达成

• 升级与活动奖励

2. 装修页面（独立场景，不影响主棋盘）

• 纯装饰养成，无任何数值加成，不加产量、不加金币

• 解锁内容：外墙、地板、门窗、路灯、庭院、家具、摆件、灯光、地毯、主题套装

3. 装修规则（有限自由度设计）

• 花店场景分为10个固定装饰槽位：
门面、橱窗、地板、墙面、吊灯、柜台区、休息区、花架区、庭院、招牌

• 每个槽位有多个可解锁的装饰选项（如"花架区"可选：简约木架/铁艺花架/旋转花柱/梦幻水晶架）

• 花愿足够即可解锁装饰选项，解锁后可自由切换已解锁的装饰

• 玩家可在槽位内自由搭配组合，打造个人风格花店

• 设计说明：相比完全无自由度的自动升级，槽位选择制让玩家有"我的花店"的归属感，同时避免了自由拖拽的开发复杂度

• 全收集驱动，长线留存核心玩法

六、玩法四：花露换装系统（独立货币，无分配焦虑）

八、建筑合成线与建筑获取节奏（合成获取，非金币购买）

设计理念：参考四季物语，建筑不再通过金币直接购买，而是通过独立的建筑合成线合成获得。玩家通过宝箱、锁定格隐藏物、任务奖励等途径获得建筑碎片/材料，在棋盘上拖拽合成升级，最终合成出功能性建筑。这一机制让建筑获取本身也成为一种合成乐趣，而非简单的"攒金币点购买"。

1. 花束建筑合成线（4级合成链 → 4种永久建筑）

花束蓝图碎片（1级）+ 花束蓝图碎片 → 花艺工具箱（2级）→ 自动转化为「花艺操作台」
花艺工具箱（2级）+ 花艺工具箱 → 花架组件（3级）→ 自动转化为「包装台」
花架组件（3级）+ 花架组件 → 花房设计图（4级）→ 自动转化为「小型温室」
花房设计图（4级）+ 花房设计图 → 自动转化为「星光花房」（顶级）

2. 饮品建筑合成线（3级合成链 → 3种永久建筑）

饮品蓝图碎片（1级）+ 饮品蓝图碎片 → 调饮器具箱（2级）→ 自动转化为「简易茶台」
调饮器具箱（2级）+ 调饮器具箱 → 吧台设计图（3级）→ 自动转化为「调饮吧台」
吧台设计图（3级）+ 吧台设计图 → 自动转化为「花饮工坊」（顶级）

3. 建筑获取节奏设计（与游戏进程对应）

• 0~5分钟（新手期）：主线任务直接奖励2个花束蓝图碎片，引导合成出首个花艺操作台
• 5~10分钟：通过做订单获得铜宝箱，开出更多花束蓝图碎片，积累建筑材料
• 10~15分钟：主线任务奖励2个饮品蓝图碎片，引导合成出首个简易茶台，同时解锁饮品品类和组合订单
• 15~30分钟：通过持续做订单和开宝箱，积累中级建筑材料（花艺工具箱/调饮器具箱）
• 30分钟~2小时：合成出包装台和调饮吧台（中级建筑），产出能力显著提升
• 2小时~1天：积累高级建筑材料，合成小型温室和花饮工坊
• 1~3天：合成顶级建筑星光花房，达到最大产出能力

4. 消耗型建筑获取

消耗型建筑不需要通过合成线获得，而是直接从以下途径获取：
• 宝箱开出（银宝箱和金宝箱有概率直接产出消耗型建筑）
• 任务奖励
• 活动奖励
• 棋盘锁定格中预置

5. 设计说明

• 建筑合成线与花束/花饮合成线并行，棋盘上同时存在三类可合成物品（花束、花饮、建筑材料），丰富合成策略
• 建筑材料占据棋盘格子，玩家需要在"存材料合建筑"和"腾空间做订单"之间做取舍
• 主线任务在关键节点直接给予建筑碎片，确保新手流畅体验首个建筑合成，不会卡在"凑不齐碎片"的状态
• 宝箱作为建筑碎片的持续来源，驱动玩家做更多订单来获得更多宝箱
• 消耗型建筑作为惊喜奖励从宝箱中产出，给玩家"开出好东西"的满足感

九、图鉴系统（全收集驱动）

1. 花束图鉴：3大花系×6级=18种花束全收集

2. 花饮图鉴：3条饮品线×3级=9种花饮全收集

3. 客人图鉴：接待全部16种客人解锁

4. 熟客图鉴：8位熟客好感度满级解锁完整故事

5. 建筑图鉴：合成全部永久型建筑（花束建筑4种+饮品建筑3种=7种永久建筑）+ 消耗型建筑6种 = 13种建筑

6. 装修图鉴：解锁全部家具装饰

7. 时装图鉴：解锁全部服装套装

8. 组合搭配成就：完成特定花束+花饮组合订单解锁（如"完美下午茶"=任意高级花束+限定手作茶）

集齐奖励：花愿、花露、钻石、限定花、限定时装

十、任务系统

1. 主线任务：引导合成建筑、解锁花系/饮品线、装修、换装、社交

2. 每日任务：合成、交付订单、获得花愿/花露、登录、互访好友、消耗体力

3. 成就任务：总订单数、收集进度、装修度、时装度、熟客好感度、建筑合成进度

十一、新手引导节奏（前30分钟体验设计，小游戏生死线）

1. 第一阶段：0~2分钟（学会合成，立刻上手）

• 开场：温馨动画，"欢迎来到你的花店"

• 引导拖拽两朵小雏菊合成向日葵，立刻产生视觉反馈

• 再合成一次，得到康乃馨

• 核心目标：让玩家2分钟内理解合成玩法并感到"好玩"

2. 第二阶段：2~5分钟（第一位客人，体验自动锁定交付）

• 第一位客人（学生少女）上门，头顶显示需要"向日葵"（单品需求）

• 棋盘上的向日葵自动被锁定（显示锁定光效），引导文字解释"花朵已为客人准备好"

• 客人旁边出现"完成"按钮，引导玩家点击

• 点击后交付完成，金币+花愿到手

• 弹出简短祝贺，展示金币用途

• 核心目标：建立"合成→自动锁定→点击完成→获得奖励"的基础循环认知

3. 第三阶段：5~10分钟（合成首个花束建筑，体验建筑产出）

• 主线任务直接奖励2个花束蓝图碎片，引导玩家在棋盘上合成出花艺操作台（首个永久型建筑）

• 教会点击建筑产花（消耗体力），体验建筑产出+CD冷却的机制

• 介绍体力系统（"点击建筑需要消耗体力哦，体力会随时间恢复"）

• 再接待2~3位客人（均为单品需求），熟悉经营节奏

• 核心目标：体验"建筑产花+合成+自动锁定+点击完成交付"完整循环，理解体力和CD机制

4. 第四阶段：10~15分钟（合成饮品建筑，体验组合订单）

• 主线任务奖励2个饮品蓝图碎片，引导合成"简易茶台"（首个饮品建筑）

• 教会点击茶台产出花饮，体验新品类

• 紧接着一位"文艺青年"上门，头顶气泡显示2个需求槽位：花草茶×1 + 粉玫瑰×1

• 引导文字解释"这位客人想要花和饮品的组合哦"

• 棋盘上花草茶先被锁定（第一个槽位亮起），玩家合成粉玫瑰后也被锁定（第二个槽位亮起）

• 全部满足后客人旁边出现"完成"按钮，点击交付，获得更丰厚的组合订单奖励

• 核心目标：理解组合订单机制，感受"凑齐一套"的满足感

5. 第五阶段：15~20分钟（解锁装修，初见花店变美）

• 花愿足够，引导进入装修页面

• 免费解锁第一件装饰（门面升级），花店外观立刻变化

• 核心目标：感受花店"变美"的成就感，建立长线养成期待

6. 第六阶段：20~30分钟（解锁换装，店主形象焕新）

• 花露足够，引导进入换装页面

• 免费获得第一套服装（清新裙装），店主形象立刻改变

• 引导截图分享

• 核心目标：所有核心玩法全部体验完毕（合成+花束交付+饮品建筑+组合订单+装修+换装），玩家已形成完整认知和留存动力

7. 设计原则

• 前5分钟决定玩家是否留下，必须在5分钟内体验到完整的"合成→自动锁定→点击完成→赚钱"循环

• 10~15分钟是第二个关键节点：解锁饮品建筑+组合订单，玩法层次跳跃式提升，给玩家"原来还有这些"的惊喜感

• 前30分钟必须让玩家体验到所有核心玩法的"爽感"

• 每个阶段结束都有明确的正反馈（动画、音效、奖励弹窗）

• 新手引导结束后，自然过渡到自由经营模式

十二、开店仪式与日结算（情感体验增强）

1. 每日开店仪式

• 每天首次登录：播放"拉开卷帘门"小动画（约2秒）

• 门面展示当前装修风格，店主穿着当前时装迎客

• 显示"今日天气"（纯装饰，随机晴/阴/雨/雪，影响花店氛围光效）

• 开店后第一位客人立刻到来，给予即时互动感

2. 每日打烊结算

• 每日首次登录时先展示昨日结算（若非首日）：

今日营业额：XXX金币
服务客人：XX位
好评率：XX%（基于交付速度和花等级匹配度）
获得花愿：XX / 获得花露：XX
熟客好感进度提示

• 好评率连续3天90%以上，奖励额外花愿/花露

• 设计说明：低开发成本，但给玩家强烈的"经营一家店"的代入感和成就感

十三、核心数值体系（稳定、不卡关、适配小游戏）

1. 合成时间：即时合成，无等待（拖拽即完成）

2. 建筑产出：
• 永久型建筑：点击产出后进入CD冷却期（30秒~10分钟不等），冷却结束后可再次点击
• 消耗型建筑：无CD，但有总使用次数限制（3~5次），用完自动消失
• 所有建筑点击均消耗体力

3. 客人刷新间隔：10~30秒

4. 体力系统（核心节奏调控资源）：

• 体力上限：100点（后期可通过成就提升至120/150）
• 自然恢复：每3分钟恢复1点体力（每小时恢复20点）
• 体力消耗项：
  ◦ 点击永久型建筑：3~12体力/次（根据建筑等级）
  ◦ 点击消耗型建筑：2~6体力/次（根据建筑等级）
  ◦ 点击宝箱：2体力/次
• 体力恢复途径：
  ◦ 自然恢复（基础，每3分钟1点）
  ◦ 金币购买体力药水：50金币恢复20体力，100金币恢复50体力（每日限购5次）
  ◦ 观看广告恢复：每次恢复30体力（每日限3次）
  ◦ 升级奖励：每次升级全额恢复体力
  ◦ 每日登录奖励：赠送30体力
  ◦ 好友互助：好友浇花赠送5体力（每日上限15体力）
• 体力为0时：建筑和宝箱不可点击，但合成操作、订单交付、挂机产出不受影响
• 设计目的：
  ◦ 控制游戏节奏，避免玩家无限点击建筑导致物品泛滥
  ◦ 为金币和广告创造消耗/变现出口
  ◦ 引导玩家合理安排每次游戏时长（约15~25分钟消耗完一轮体力）
  ◦ 配合离线回复，形成"回来就有体力可以操作"的登录动力

5. 挂机产出：
• 持续生成基础物品（1级花束和1级花饮），花束三个花系随机产出，花饮三条饮品线随机产出
• 花饮挂机产出需解锁首个饮品建筑后开启
• 存储上限：根据棋盘格子数计算，满格后停止产出
• 产出速度：基础1个/60秒，可通过金币提速

5. 离线收益：
• 支持离线产出，上限为2~4小时的产出量
• 超过4小时不再累积，引导玩家每天至少登录2次
• 广告可翻倍已有离线收益（不突破时间上限，只翻倍数量）
• 离线收益领取时播放"开店清点"小动画

核心元素：
1. 店主形象（最重要的展示位）

   • 尺寸：约区域B高度的70%，Q版半身像
   • 展示内容：实时反映当前换装搭配（发型、服装、饰品全部显示）
   • 交互：点击店主形象 → 弹出快捷菜单：「换装」「查看形象」
   • 动画：待机状态下有轻微的呼吸动画/眨眼动画（2~3帧循环）
   • 设计说明：换装系统的全部意义在于"让玩家在最常看到的页面欣赏自己搭配的角色"，所以店主必须在主页面以醒目尺寸展示

2. 客人展示区

     ◦ 气泡右侧小字显示奖励预览（金币+花愿或花露图标）
     ◦ 如果是熟客，名字下方显示小爱心图标+当前好感度等级
   • 客人动画：走入→站定→等待（轻微晃动）→全部槽位满足后出现"完成"按钮→点击后满意离开
   • 交付方式：自动锁定+手动确认（棋盘上匹配物品自动锁定，全部满足后客人旁边出现"完成"按钮，点击交付）

--- 区域C：合成棋盘区 ---

• 位置：花店场景下方，占屏幕高度约50%（核心操作区，占比最大）
• 背景：温暖的木质台面纹理（与花店风格统一）

核心元素：
1. 网格棋盘

   • 扩容后最大：6列×5行 = 30格
   • 每格尺寸：正方形，约屏幕宽度的1/6.5（留边距）
   • 格子样式：圆角方格，浅米色底
  • 锁定格子类型：迷雾格（灰色遮罩+锁头）、可窥视格（半透明遮罩+物品可见+可跨格合成）、钥匙格（金色锁头+解锁价格）
  • 详细解锁规则见"棋盘格渐进解锁系统"章节
   • 格子内容：鲜花物品 或 生产建筑 或 空格

2. 鲜花/花饮物品（在格子内）
   • 物品图标居中显示，右下角小数字标示等级
   • 花束类视觉区分：
     ◦ 日常花系：图标底部有淡黄绿色光晕
     ◦ 浪漫花系：图标底部有淡粉紫色光晕
     ◦ 奢华花系：图标底部有淡金色光晕
   • 花饮类视觉区分：
     ◦ 茶饮线：图标底部有淡茶色光晕，杯具/茶壶造型
     ◦ 冷饮线：图标底部有淡冰蓝色光晕，冷杯/气泡造型
     ◦ 甜品线：图标底部有淡粉橙色光晕，蛋糕/甜品造型
   • 拖拽交互：
     ◦ 长按物品 → 物品放大浮起 → 拖拽到同品类同系同等级物品上 → 合成动画（闪光+粒子）→ 生成高一级物品
     ◦ 拖拽到不匹配的物品上 → 轻微抖动+红色X提示 → 物品归位
     ◦ 花束和花饮不可跨品类合成
   • 锁定状态（自动锁定机制）：
     ◦ 被锁定的物品显示锁定特效（柔和的金色边框持续闪烁+客人头像小标记）
     ◦ 锁定物品仍然可以被拖拽合成——增加策略深度，玩家可以选择把锁定物品拿去合成更高级的
     ◦ 如果锁定物品被合成消耗，对应客人的需求槽位锁定自动取消，系统重新扫描匹配
     ◦ 锁定特效提醒让玩家清楚知道哪些物品已被预定给客人，合成时心中有数
   • 新合成的物品有3秒金色边框闪烁，提示玩家注意

3. 生产建筑（在格子内，占1格，分花束建筑和饮品建筑两类，分永久型和消耗型）
   • 建筑图标比普通物品略大（占满整格），有独特的建筑外观
   • 花束建筑以花朵/花架元素为主视觉，饮品建筑以杯具/茶壶元素为主视觉
   • 永久型建筑状态显示：
     ◦ 可点击状态：建筑图标正常亮度，轻微呼吸光效，右上角绿色圆点
     ◦ CD冷却状态：建筑图标半透明灰暗，右上角倒计时数字，底部进度条
     ◦ CD冷却结束：建筑恢复亮度+闪光提示+音效"叮"
   • 消耗型建筑状态显示：
     ◦ 建筑图标上方显示剩余使用次数（如"3/5"）
     ◦ 每次使用后次数减少，最后一次使用时图标闪烁提示"最后一次"
     ◦ 用尽后建筑播放消散动画（碎片飞散），释放格子
   • 点击交互：
     ◦ 花艺操作台/简易茶台（初级）：点击 → 检查体力是否足够 → 产出对应品类物品到相邻空格 → 扣除体力 → 永久型进入CD
     ◦ 中高级建筑：点击 → 弹出产出线选择浮层（花系/饮品线图标按钮）→ 选择后检查体力 → 产出到相邻空格 → 扣除体力
     ◦ 产出后播放弹跳动画反馈
     ◦ 如果体力不足 → 提示"体力不足"+ 弹出体力恢复选项（金币购买/广告恢复）
     ◦ 如果相邻无空格 → 提示"棋盘已满，请先合成"

4. 宝箱物品（在格子内，占1格，消耗型）
   • 宝箱图标根据等级显示不同外观（铜色/银色/金色）
   • 右上角显示剩余可点击次数
   • 点击交互：每次点击消耗体力，产出建筑碎片/消耗型建筑/金币/花愿/花露
   • 产出物品落入相邻空格
   • 所有次数用尽后宝箱自动消失

   • 花束和花饮的产出动画略有区别：花束带花瓣飘散效果，花饮带气泡冒出效果
   • 棋盘满格时，右上角出现"棋盘已满"红色小提示

十八、棋盘格渐进解锁系统（迷雾探索机制）

设计理念：参考四季物语，棋盘格并非一开始全部开放，而是从中心向外逐步解锁。未解锁的格子处于"迷雾/锁定"状态，部分锁定格子内隐藏着固定的高等级物品，玩家通过合成来"揭开"周围格子。这一机制为棋盘增加了探索感和成就感，同时自然控制了游戏节奏。

1. 棋盘初始布局

• 棋盘最大规模：6列×5行 = 30格（与原设计一致）
• 初始开放区域：中心2列×2行 = 4格（最小可玩区域）
• 初始开放格中预置：2个1级日常花束 + 1个花材礼盒（初级消耗型建筑，供新手体验建筑点击产出）+ 1个空格
• 主线任务在前5分钟内奖励2个花束蓝图碎片，引导合成首个花艺操作台
• 其余26格全部为锁定状态，分布在初始区域周围

2. 锁定格子的状态分类

锁定格子分为三种类型：

（1）迷雾格（不可见内容）
• 视觉表现：灰色半透明遮罩 + 锁头图标 + 模糊背景
• 格子内可能有物品也可能为空，但玩家无法看到内容
• 必须通过相邻已解锁格子中的合成操作来解锁

（2）可窥视格（内容可见，不可操作）
• 视觉表现：轻度半透明遮罩 + 物品清晰可见 + 格子边框发光提示
• 格子内的物品可以看到，但不可拖动、不可点击
• 关键机制：如果玩家在已解锁的相邻格子中合成了一个物品，且该物品与可窥视格中的物品满足合成条件（同品类+同系+同等级），则触发"跨格合成"——两个物品合成为高一级物品，合成结果留在已解锁格子中，可窥视格永久解锁变为空格
• 这是核心策略点：玩家可以看到锁定格中的物品，需要主动合成对应等级的物品来与之"对碰"解锁

（3）钥匙格（需要消耗金币/道具解锁）
• 视觉表现：金色锁头 + 显示解锁所需金币数量
• 直接消耗金币即可解锁，解锁后显示内容（可能是高级物品、建筑或空格）
• 分布在棋盘边缘或关键位置，作为金币消耗出口

3. 锁定格解锁规则

（1）相邻合成解锁（核心解锁方式）
• 当玩家在已解锁格子中成功合成一个新物品时，该格子周围的迷雾格会被"波及"解锁
• 解锁波及范围：合成发生的格子上下左右4个相邻格（不含对角线）
• 每次合成解锁1~2个相邻迷雾格（取决于合成产物等级）：
  ◦ 合成产物等级1~2：解锁1个相邻迷雾格
  ◦ 合成产物等级3~4：解锁1~2个相邻迷雾格
  ◦ 合成产物等级5~6：解锁2个相邻迷雾格
• 如果相邻没有迷雾格可解锁，则无额外效果

（2）跨格合成解锁（可窥视格专属）
• 当玩家在已解锁格子中持有一个物品，且相邻的可窥视格中有一个同品类、同系、同等级的物品时：
  ◦ 可窥视格中的物品边框会闪烁提示"可合成"
  ◦ 玩家将自己的物品拖动到可窥视格上，触发合成
  ◦ 合成结果（高一级物品）出现在玩家原来的格子中（或最近的空格）
  ◦ 可窥视格永久解锁，变为空格
  ◦ 解锁时同时波及该格周围的迷雾格，将1~2个迷雾格转变为可窥视格或直接解锁

（3）连锁解锁（惊喜机制）
• 当一个迷雾格被解锁后，如果其中隐藏的物品与相邻已解锁格中的物品满足合成条件，系统不会自动合成，但会高亮提示"可合成"
• 如果一个可窥视格被合成解锁后，波及解锁了新的格子，新格子中又有可与现有物品合成的内容，则形成"连锁解锁"的探索兴奋感

4. 锁定格内预置物品规则

（1）预置物品设计原则
• 锁定格中的物品等级通常高于玩家当前阶段能轻松合成的等级，形成"够一够"的目标感
• 预置物品是固定的（每个锁定格的内容在棋盘布局数据中预设好），不是随机生成的
• 预置物品覆盖花束和花饮两个品类，引导玩家探索不同合成路线

（2）预置物品分布参考（30格棋盘示例）

棋盘格坐标（行,列）从左上角(0,0)开始：

第一圈解锁区（初始4格周围的8格，预计5~15分钟内解锁）：
• 以迷雾格和可窥视格混合分布
• 可窥视格中放置：2级日常花束、2级浪漫花束等
• 迷雾格中隐藏：空格、1级花束、花束蓝图碎片
• 目的：教会玩家"跨格合成"的核心操作，同时提供建筑碎片加速建筑合成

第二圈解锁区（外围10~12格，预计15~45分钟解锁）：
• 可窥视格中放置：3~4级花束、2级花饮（引导解锁饮品线）
• 迷雾格中隐藏：花艺工具箱（2级建筑材料）、金币奖励、2~3级花束、消耗型建筑
• 钥匙格出现（需要金币解锁）
• 目的：引导玩家接触中级物品、开始花饮品类，获得中级建筑材料

第三圈解锁区（最外围6~8格，预计1~3小时解锁）：
• 可窥视格中放置：4~5级高级花束、3级顶级花饮
• 迷雾格中隐藏：高级建筑材料（花架组件/吧台设计图）、消耗型建筑（高级）、稀有物品
• 钥匙格分布更多（金币消耗增加）
• 目的：长线目标，给玩家持续探索动力，同时为高级建筑合成提供材料

5. 视觉表现与动画

（1）迷雾格视觉
• 底色：深灰色半透明遮罩（opacity 0.7）
• 中心：锁头图标（小锁+轻微呼吸动画）
• 边缘：与已解锁格交界处有柔和的迷雾渐变效果
• 鼠标/手指移到迷雾格上时：锁头轻微晃动

（2）可窥视格视觉
• 底色：浅灰半透明遮罩（opacity 0.3），物品清晰可见
• 物品上方有细小的锁链图标，表示"被锁住了"
• 当相邻有可合成的物品时：格子边框金色闪烁 + 物品轻微上下浮动

（3）解锁动画（核心爽感来源）
• 迷雾消散：迷雾从中心向四周扩散消失（约0.8秒），伴随清脆的"叮~"音效
• 物品揭示：如果格子内有物品，物品从缩小状态弹出放大到正常尺寸（弹性动画）
• 连锁波及：解锁的波及效果像水波纹一样向外扩散，每个被波及的迷雾格有轻微闪动
• 跨格合成：物品从可窥视格飞向已解锁格，合成闪光特效后生成高级物品

（4）钥匙格解锁
• 点击钥匙格弹出确认框："消耗 XX 金币解锁此格？"
• 确认后金色锁头旋转打开→金光四射→格子内容显现

6. 棋盘布局数据结构

每个格子的预设数据包含：
• 坐标（行, 列）
• 初始状态：open（开放）/ fog（迷雾）/ peek（可窥视）/ key（钥匙格）
• 预置内容：empty（空）/ item:物品ID / building:建筑ID
• 钥匙格解锁价格（仅钥匙格有）
• 解锁优先级（控制波及解锁时优先解锁哪个方向的格子）

7. 与现有系统的交互

（1）与建筑系统的关系
• 部分锁定格中预置建筑碎片/建筑材料，玩家解锁格子后获得这些材料用于合成建筑
• 消耗型建筑也可能预置在锁定格中，解锁后直接可用
• 建筑通过合成线合成获得，不再通过金币购买
• 保留部分钥匙格用于放置高级建筑材料，作为金币消耗出口

（2）与客人系统的关系
• 客人需求扫描只扫描已解锁格子中的物品
• 可窥视格中的物品不参与自动锁定匹配（因为无法拖动）

（3）与挂机系统的关系
• 挂机产出只能放入已解锁的空格中
• 如果已解锁格全满，挂机产出暂停（与原设计一致）

（4）与新手引导的整合
• 阶段1（0~2分钟）：在初始4格中学会基础合成
• 阶段1.5（2~5分钟）：引导玩家合成一个2级花束，触发首次迷雾格解锁，体验解锁动画的爽感
• 阶段2（5~8分钟）：引导玩家发现可窥视格，教会"跨格合成"操作
• 后续阶段：自然融入，不再强制引导

8. 数值平衡要点

• 初始4格足够完成基础合成学习和首次客人交付
• 前15分钟内应能解锁到8~10格，保证有足够操作空间
• 可窥视格中的物品等级不应太高（前期2~3级），避免玩家"够不到"的挫败感
• 钥匙格的金币价格应与正常经营收入匹配：前期格子50~100金币，后期格子500~2000金币
• 最终全部解锁30格应在正常游戏3~5天内完成（非肝玩家）

9. 设计目的总结

• 探索感：每次解锁新格子都有"开盲盒"的惊喜感，尤其是迷雾格内隐藏的高级物品
• 策略性：可窥视格的"跨格合成"机制要求玩家有目标地合成特定等级物品，增加合成策略深度
• 节奏控制：通过格子逐步解锁自然控制玩家获得建筑和高级物品的节奏，避免一开始就内容爆炸
• 成就感：从4格小棋盘一步步扩展到30格大棋盘，花店"越来越大"的视觉成长感
• 金币消耗：钥匙格提供金币消耗出口，与合成经营的金币产出形成健康循环

二十、限定内容产出规则（含限时活动+限定花/时装/装扮）

1. 限定内容范围

包括：限定花束、限定时装、限定花店装修、限定背景、限定店主皮肤、限定摆件。

2. 三套货币完全区分（数值健康、不混乱、不通胀）

1）日常货币一：花愿
• 来源：日常订单、交付客人、任务、成就
• 用途：兑换永久普通装修

2）日常货币二：花露
• 来源：交付客人掉落、熟客订单、任务、成就、好友互访
• 用途：兑换永久普通时装

3）限时活动货币：活动花瓣（节日/版本专属）
• 来源：仅限时活动期间产出
• 用途：兑换全部限定奖励（限定花、限定时装、限定装修）

3. 限定内容唯一来源：限时活动玩法（对标四季物语活动机制）

1. 活动开启后，主棋盘随机产出活动专属花（如：情人节玫瑰、中秋月桂、圣诞雪花花、春日樱花）

2. 活动花可在棋盘内拖拽合成更高阶活动花（活动花独立短链，4级）

3. 活动花可交付给活动专属客人/活动订单

4. 完成后获得：活动花瓣

5. 使用活动花瓣在活动商店兑换：

◦ 限定花束（永久收藏，活动结束后暂绝版）

◦ 限定时装套装

◦ 限定花店装修皮肤

◦ 限定背景/限定特效

4. 活动花与日常花的关系

• 活动花不参与日常合成链

• 活动花不能用于日常订单

• 活动花仅在活动期间存在，活动结束后不再产出

• 已获得的限定花、时装、装修永久保留

5. 限定产出核心规则

• 日常内容：永久可获取、无压力收集

• 限定内容：仅限时活动产出，不进入常规养成

• 活动结束：限定内容暂绝版，后期可节日返场

• 无强制氪金，主要靠活动玩法肝取，支持少量礼包辅助获取

6. 活动设计目的（长期留存、版本迭代、商业化）

• 给玩家阶段性目标，提升长期留存

• 每月/节日更新版本，保持游戏新鲜感

• 限定外观满足收集欲、晒图分享、社交传播

• 支持商业化礼包（限定礼包、快速兑换礼包），不破坏平民体验

二十二、长线扩展方向规划（装修+换装内容池扩展）

设计背景：当前装修系统（10槽位自由搭配）和换装系统（多部件分层搭配）相比四季物语的固定线性解锁，具备更强的个性化和归属感，但内容池集中在一个花店场景+一个店主角色，需要预留长线扩展空间，避免内容见顶。

以下三个方向可叠加使用，按推荐优先级排序：

1. 主题季/主题包（优先级最高，上线后即可启动）

• 不增加新场景，为同一个花店持续推出主题皮肤包
• 每个主题包包含：一套主题装饰（10个槽位各1~2个新选项）+ 主题时装套装 + 限定花系（活动花束合成链）+ 限定客人（主题客人带特殊订单）
• 示例：「春日樱花季」「圣诞暖冬」「万圣奇妙夜」「新年花市」「夏日海滨」等
• 优点：不改系统架构，纯内容扩展（策划出配置+美术出素材），内容池开放式增长
• 节奏：上线后每月一个主题包，与现实节日/季节联动

2. 多区域/多楼层（中期扩展，1.x版本）

• 花店扩展出新区域，每个区域复用同一套槽位装修系统，但有独立的槽位定义和装饰选项
• 预想区域：
  ◦ 花店一楼（当前主区域，已实现）
  ◦ 花店二楼/阁楼（私人空间——卧室+休息区）
  ◦ 后花园/温室
  ◦ 分店（不同城市风格——法式花店/日式花道馆）
• 新区域通过花愿或主线任务解锁，每个区域带来全新装饰收集目标
• 优点：系统架构完全复用，开发成本为场景美术+装饰配置，兼具四季物语的场景扩展性和我们的自由搭配特色

3. NPC/熟客故事线房间（长线扩展，2.x版本）

• 利用已有的8位熟客系统做深度扩展
• 熟客好感度满级后解锁「帮TA布置房间」玩法：每位熟客有独立场景（学生少女的宿舍/文艺青年的书房/退休教授的书房…），同样采用槽位装修制
• 扩展后熟客系统变为：刷好感度 → 看故事 → 帮TA装修房间 → 解锁更多故事
• 优点：叙事+装修深度结合，随NPC数量自然扩展，情感粘性极强
• 新增NPC可在后续版本持续追加，每位NPC = 1条故事线 + 1个可装修场景

扩展方向总结表：

| 扩展方向 | 开发成本 | 扩展上限 | 建议时期 |
|---------|---------|---------|---------|
| 主题季/主题包 | 低（纯配置+美术） | 无限 | 上线后每月一个 |
| 多区域/多楼层 | 中（复用系统，新场景美术） | 5~8个区域 | 1.x版本 |
| NPC故事线房间 | 中高（每位熟客×独立场景） | 随NPC数量扩展 | 2.x版本 |

核心原则：当前槽位装修制+部件换装制的架构本身具备良好扩展性，关键是为其提供更多「空间载体」（多区域/多NPC房间）和「内容注入」（主题季），三个方向可叠加实施。

二十三、最终总结

本文档为修订版完整策划案，在四季物语全部核心机制基础上，做了以下关键优化：

• 双品类设计：花束（3花系×6级=18种）+ 花饮（3饮品线×3级=9种）= 27种商品，收集驱动力翻倍
• 棋盘格渐进解锁：初始4格→逐步解锁至30格，迷雾格探索+可窥视格跨格合成+钥匙格金币解锁，自然控制节奏并增加探索乐趣
• 花束合成链拆分为3条短链（每系6级），花饮为3条超短链（每线3级），长短互补节奏丰富
• 双类型建筑：永久型建筑（CD冷却，长线稳定产出）+ 消耗型建筑（用完消失，短期爆发），通过建筑合成线合成获得而非金币购买
• 体力系统：建筑和宝箱点击消耗体力，自然恢复+金币购买+广告恢复，调控游戏节奏
• 宝箱系统：铜/银/金三级宝箱，可合成升级，产出建筑碎片和消耗型建筑，驱动订单→宝箱→建筑的正循环
• 组合订单系统：客人从"买一种"升级为"买一套"（1~3个需求槽位），花束+花饮搭配购买
• 自动锁定+手动确认交付机制：棋盘物品自动匹配客人需求并锁定（锁定物品仍可合成增加策略性），凑齐后点击"完成"按钮交付
• 货币拆分为花愿（装修）+花露（换装），消除分配焦虑
• 新增熟客养成系统（8位熟客，好感度5级，含故事解锁），增强情感粘性
• 装修改为槽位自由搭配制（10个槽位），兼顾自由度与开发成本
• 新增轻社交系统（互访/助力/排行榜），适配微信生态传播
• 新增开店仪式+日结算，增强经营代入感
• 明确新手前30分钟引导节奏：0~10分钟花束入门→10~15分钟解锁饮品+组合订单→15~30分钟装修换装全体验
• 离线收益上限2~4小时，引导健康登录频率
• 新增完整界面设计规范（5个核心页面+弹窗+交互细节+跳转逻辑），可直接交付UI/UE设计

新增内容所需额外美术资源：9种花饮商品素材 + 3种饮品建筑素材 + 6种消耗型建筑素材 + 3级宝箱素材 + 建筑合成线材料素材（花束4级+饮品3级）+ 体力UI图标，开发成本可控。

可直接交付程序、美术、UI、运营进行开发。

# 游戏资产：AI 生图与入库流程（花语小筑）

本流程与 Cursor 规则 `.cursor/rules/game-art-asset-flow.mdc` 一致，便于人工与 Agent 重复执行。

若自动化脚本无法写入仓库外的 `game_assets`（权限限制），在终端手动执行：

`cp .../assets/raw/house_yard_bg_nb2.png .../game_assets/huahua/assets/house/bg.png`

## 1. 提示词

- 写入仓库内：`docs/prompt/{任务名}_nb2_prompt.txt`（**文件内仅英文正文**，无 `#` 注释，整文件会原样送给 API）
- 花店室外草地背景：`docs/prompt/house_yard_bg_nb2_prompt.txt` → 原图 `house_yard_bg_nb2.png`

## 2. 生图（NB2）

使用 [gemini-image-gen](https://github.com/) skill 脚本，**默认模型即 NB2**：

```bash
python3 ~/.cursor/skills/gemini-image-gen/scripts/generate_images.py \
  --prompt-file /Users/huyi/rosa_games/huahua/docs/prompt/house_yard_bg_nb2_prompt.txt \
  --output /Users/huyi/rosa_games/game_assets/huahua/assets/raw/house_yard_bg_nb2.png \
  --model gemini-3.1-flash-image-preview \
  --aspect-ratio 9:16
```

- 需要可访问 Google API（本机代理见 skill 说明）。
- **原图**统一落在：`/Users/huyi/rosa_games/game_assets/huahua/assets/raw/`

## 3. 拷贝进游戏工程

- 运行时资源路径（与 `TextureCache.house_bg` 一致）：

```bash
cp /Users/huyi/rosa_games/game_assets/huahua/assets/raw/house_yard_bg_nb2.png \
   /Users/huyi/rosa_games/huahua/minigame/subpkg_deco/images/house/bg.png
```

- 若需与旧版 `sync-assets.sh` 的 `house/bg.png` 约定一致，可额外保留一份：

```bash
mkdir -p /Users/huyi/rosa_games/game_assets/huahua/assets/house
cp /Users/huyi/rosa_games/game_assets/huahua/assets/raw/house_yard_bg_nb2.png \
   /Users/huyi/rosa_games/game_assets/huahua/assets/house/bg.png
```

- 可选：将宽边缩至 750px（与历史同步脚本一致）：

```bash
sips -Z 750 /Users/huyi/rosa_games/huahua/minigame/subpkg_deco/images/house/bg.png
```

## 4. 试看图（不替换正式资源）

花店房间壳 `bg_room_*.png` 改版可先输出到资产库预览目录，并在仓库 `house/preview/` 留一份方便对比：

```bash
mkdir -p /Users/huyi/rosa_games/game_assets/huahua/assets/preview_house_room

python3 ~/.cursor/skills/gemini-image-gen/scripts/generate_images.py \
  --prompt-file "{workspace}/docs/prompt/house_bg_room_default_v2_nb2_prompt.txt" \
  --output "/Users/huyi/rosa_games/game_assets/huahua/assets/preview_house_room/bg_room_default_v2_nb2.png" \
  --model gemini-3.1-flash-image-preview \
  --aspect-ratio 1:1

cp .../preview_house_room/bg_room_default_v2_nb2.png \
   "{workspace}/minigame/subpkg_deco/images/house/preview/"
```

验收后再 `cp` 覆盖 `subpkg_deco/images/house/bg_room_default.png`。

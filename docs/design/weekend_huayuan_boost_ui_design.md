# 周末花愿加成活动 — UI 设计说明（原型阶段）

## 活动规则（与代码一致）

| 项 | 说明 |
|---|---|
| 开放 | 仅周六、周日顶栏显示入口 |
| 权益 | 当日所有客人订单花愿 **+50%**（单独第二颗胶囊展示加成） |
| 激活 | 观看 1 次激励视频，每日 1 次 |
| 重置 | 本地日期跨日 0 点失效 |

## 1. 顶栏活动图标（替换当前纯色块）

**尺寸**：约 62×46 逻辑像素（可入库 128×128 或 256×256 透明 PNG，等比缩放）。

**造型**：
- 圆角方形徽章 + 轻微外发光，像「限时礼包」角标，不是普通文字按钮。
- 主色：**珊瑚橙 → 金橙渐变**（未激活，吸引点击）；激活后 **翠绿 + 金色勾/光晕**。
- 中心：**樱花/花愿图标** + 缎带角标写 **「周末」**；底部条幅 **「+50%」** 用特大号黄字+深描边。
- 装饰：小星星、花瓣、沙漏/日历「六日」暗示（图形化，避免小字糊）。
- 气质：与顶栏商店 NB2 图标同级精致度，**高饱和 pastel**，软 3D 贴纸感。

**状态**（工程用 PIXI 叠字，原型可画在图上参考）：
- 未激活：`周末` + `+50%`
- 已激活：`已加成` + `+50%`（整体偏绿）

## 2. 点击后 — 宣传页（非传统方框弹窗）

**结构（自上而下）**：

```
┌─────────────────────────────┐
│  [可选] 右上红底圆关闭钮      │  ← 与全局面板规范一致，略压角
├─────────────────────────────┤
│                             │
│   全幅竖版宣传插画主图区      │  ← 9:16 画布的主体 70%～78%
│   · 周末限时横幅/日历丝带     │
│   · 超大「花愿 +50%」艺术字   │
│   · 花店女主/花束/金币花瓣雨  │
│   · 「超值」「仅限周末」卖点  │
│                             │
├─────────────────────────────┤
│  胶囊主按钮（宽 80% 居中）    │  ← 工程 PIXI.Text 叠字
│  未激活：看广告 0/1           │
│  已激活：当日已获得（灰/绿）   │
└─────────────────────────────┘
```

**主图内容要点**：
- 标题区：「周末限时」烫金/丝带 banner。
- 核心卖点：**「花愿获得提升 50%」** 最大字号（可拆两行：花愿获得提升 / **50%**）。
- 副文案：「看广告激活 · 今日订单全享」「超值福利 · 次日 0 点重置」（小号，工程可 PIXI 覆盖）。
- 画面：温馨花店 merge 风，粉紫黄 pastel，花瓣与 +50% 光效，**不要**写实照片风。

**按钮**：
- 未激活：蜜黄/珊瑚渐变胶囊 + 播放三角小图标 + **看广告 0/1**。
- 已激活：浅灰绿底 + **当日已获得**，不可点或弱点击反馈。

**与现有 ConfirmDialog 关系**：原型阶段用整页宣传图替代简单确认框；接入时新建 `WeekendHuayuanBoostPanel` 或同类全屏层。

## 4. 接入备忘（实现阶段）

- 图标：`TextureCache` 注册 `icon_weekend_huayuan_boost`，替换 `TopBar._buildWeekendBoostButton` 的 Graphics。
- 面板：点击 `panel:openWeekendHuayuanBoost` → 打开宣传页；按钮调 `AdManager.showRewardedAd(WEEKEND_HUAYUAN_BOOST)`；成功后 `WeekendHuayuanBoostManager.activateToday()` 并刷新按钮文案。

《花语小筑》完整版策划案（修订版）

（对标四季物语全套结构 + 独家花店+换装双养成 + 多花系短链合成 + 熟客养成 + 轻社交，可直接开发）

一、项目基础信息

1. 游戏名称：花语小筑

2. 游戏类型：2D仿3D斜45°视角，合成经营+家园装修+店主换装微信小游戏

3. 对标产品：四季物语（核心机制、界面、数值、节奏对齐，并在关键环节做差异化优化）

4. 目标用户：全年龄段女性休闲玩家，喜爱治愈、收集、养成、搭配、经营

5. 美术风格：清新柔和、低饱和度、温馨明亮，纯2D资源，伪3D透视

6. 核心优势：
• 保留四季物语全部成功玩法
• 新增换装养成线，女性吸引力更强
• 多花系短链合成，策略性更丰富
• 熟客养成+轻社交，留存与传播更优

二、游戏核心五套玩法

1. 棋盘合成玩法（基础核心）

2. 客人订单经营 + 熟客养成（中期目标）

3. 花愿装修系统（长线家园收集）

4. 花露换装系统（独家时装收集）

5. 轻社交系统（微信生态传播）

三、玩法一：棋盘合成系统（多花系短链设计）

1. 棋盘规则

• 主界面下方为固定合成棋盘，拥有固定格子数（初始12格），可后期扩容

• 棋盘内包含：鲜花物品 + 生产建筑

• 支持拖拽合成、点击建筑产出

2. 拖拽合成（固定升级）

• 两朵同花系同等级鲜花拖拽合并，100%合成高一级花朵

• 无消耗、无失败、无冷却，合成带闪光特效与反馈

• 不同花系之间不可跨系合成

3. 三大花系合成链（每系6级，共18种花）

花系一：日常花系（基础产量高，满足初期订单）
1 小雏菊 → 2 向日葵 → 3 康乃馨 → 4 满天星花束 → 5 混搭花束 → 6 精致礼盒花

花系二：浪漫花系（中级品质，浪漫场景订单专用）
1 粉玫瑰 → 2 百合 → 3 郁金香 → 4 薰衣草花束 → 5 告白玫瑰礼盒 → 6 婚礼花艺

花系三：奢华花系（高级稀有，大单专用）
1 星空兰 → 2 生日花礼 → 3 星空花礼 → 4 鎏金花束 → 5 极光花礼 → 6 永恒花海典藏

合成规则：同花系N级 + N级 = N+1级

设计说明：
• 拆分为3条独立短链，避免单链18级导致的数值爆炸（2^17=131072个基础花才能合一个顶级花）
• 每条链仅需2^5=32个基础花即可合成顶级花，数值健康可控
• 不同花系对应不同建筑产出和客人需求，增加策略深度

4. 建筑产出（即时产出 + 装填选择机制）

建筑直接放置在棋盘，点击即时产出，无冷却时间。中高级建筑可选择产出花系。

建筑清单：

1. 花艺操作台
可选产出：日常花系（70%1级、25%2级、5%3级）
无需选择，固定产出日常花系

2. 包装台
装填选择：可选"日常花系"或"浪漫花系"
产出概率：60%1~2级、30%3~4级、10%5级

3. 小型温室
装填选择：可选"浪漫花系"或"奢华花系"
产出概率：55%1~2级、35%3~4级、10%5级

4. 星光花房
装填选择：可选三个花系中的任意一个
产出概率：40%3~4级、40%5级、20%6级

设计说明：
• 点击即产出，无冷却等待，操作节奏爽快流畅（对标四季物语）
• 平衡性通过"建筑占棋盘格子"来控制——建筑放多了花朵就没地方放，玩家需在建筑数量和合成空间之间取舍
• 初级建筑无选择，降低新手门槛
• 中高级建筑增加花系选择，玩家可根据当前订单需求主动规划产出
• 增加了策略深度，避免纯粹的"无脑点点点"

四、玩法二：客人订单经营系统 + 熟客养成（核心赚钱循环）

1. 客人机制

• 客人随机在主界面刷新，头顶显示需求花朵（含花系标识）

• 拖拽对应花束交付，获得金币 + 花愿/花露

• 客人等级越高，需求花越高级，奖励越高

2. 客人类型（16种）

学生少女、上班族、温柔妈妈、文艺青年、情侣、小朋友、生日顾客、告白男生、婚礼新娘、节日顾客、网红博主、贵妇、收藏家、镇长、特邀嘉宾、神秘顾客

3. 熟客养成系统（新增核心留存机制）

• 部分客人（学生少女、上班族、温柔妈妈、文艺青年、情侣、网红博主、贵妇、收藏家，共8位）为可养成熟客

• 每次交付成功，对应熟客好感度+1

• 好感度等级与奖励：

好感度Lv1（累计交付5次）：解锁客人小档案（姓名、性格、故事背景）
好感度Lv2（累计交付15次）：解锁专属订单（奖励翻倍）
好感度Lv3（累计交付30次）：解锁客人故事剧情（如"告白男生终于求婚成功"）
好感度Lv4（累计交付50次）：解锁专属头像框 + 大量花愿/花露奖励
好感度Lv5（累计交付80次）：客人外观升级 + 解锁限定图鉴

• 设计说明：熟客故事为短文本+静态插图，开发成本低但情感粘性极强，对女性用户留存效果显著

4. 订单体系

• 即时订单：客人上门即时交付

• 每日订单：固定3~5单，奖励更高

• 限时高级订单：产出钻石+大量花愿

• 熟客专属订单：好感度Lv2后解锁，奖励翻倍，可指定花系需求

5. 金币唯一用途

• 解锁棋盘内生产建筑（花艺台、温室等）

• 解锁棋盘格子数量

• 提升基础挂机产出速度

6. 经营循环

合成花 → 选择建筑产出花系 → 交付客人 → 赚金币+花愿/花露 → 解锁建筑 → 养熟客 → 更高订单

五、玩法三：花愿装修系统（对标四季物语星星家园，增加有限自由度）

1. 花愿获取途径

• 完成所有订单

• 交付客人

• 升级建筑

• 图鉴收集

• 每日任务

• 成就达成

• 升级与活动奖励

2. 装修页面（独立场景，不影响主棋盘）

• 纯装饰养成，无任何数值加成，不加产量、不加金币

• 解锁内容：外墙、地板、门窗、路灯、庭院、家具、摆件、灯光、地毯、主题套装

3. 装修规则（有限自由度设计）

• 花店场景分为10个固定装饰槽位：
门面、橱窗、地板、墙面、吊灯、柜台区、休息区、花架区、庭院、招牌

• 每个槽位有多个可解锁的装饰选项（如"花架区"可选：简约木架/铁艺花架/旋转花柱/梦幻水晶架）

• 花愿足够即可解锁装饰选项，解锁后可自由切换已解锁的装饰

• 玩家可在槽位内自由搭配组合，打造个人风格花店

• 设计说明：相比完全无自由度的自动升级，槽位选择制让玩家有"我的花店"的归属感，同时避免了自由拖拽的开发复杂度

• 全收集驱动，长线留存核心玩法

六、玩法四：花露换装系统（独立货币，无分配焦虑）

1. 换装专用货币：花露（与装修花愿独立）

• 花露来源：
◦ 交付客人时小概率掉落（约30%概率）
◦ 熟客专属订单必掉花露
◦ 每日任务奖励
◦ 成就奖励
◦ 好友互访赠送

• 设计说明：拆分为两种货币后，玩家不再有"装修还是换装"的选择焦虑，两条养成线可同步推进，体验更流畅

2. 换装内容

• 套装：日常服、花艺师工装、清新裙装、温柔系、节日装、限定主题装

• 部件：发型、发饰、耳环、包包、鞋子

• 全部为外观展示，无属性加成

3. 换装作用

• 店主形象自由搭配

• 支持截图分享、社交展示

• 满足女性玩家搭配、收集、养成的核心乐趣

• 与装修形成双养成线，极大提升长期留存

七、玩法五：轻社交系统（微信生态传播核心）

1. 好友花店互访

• 可进入好友花店参观，查看装修和店主形象

• 互访双方各获得少量花愿+花露（每日上限3次）

• 可给好友花店点赞，累计点赞数显示在花店门面

2. 好友助力

• 被帮助方收到通知，形成双向互动

3. 排行榜

• 花店美观度排行（装修收集进度）

• 收集度排行（图鉴完成度）

• 本周营业额排行

4. 合作订单（后期版本开放）

• 好友间可互赠低级花材（每日2次）

• 双人合作大订单：各出指定花系花束，共同完成，奖励更高

5. 社交分享

• 一键截图分享花店/店主形象到微信

• 分享可获得少量花愿奖励（每日上限）

• 被分享链接点击后，新用户可直接进入游戏

十、任务系统

1. 主线任务：引导解锁建筑、鲜花、装修、换装、社交

2. 每日任务：合成、交付订单、获得花愿/花露、登录、互访好友

十一、新手引导节奏（前30分钟体验设计，小游戏生死线）

1. 第一阶段：0~2分钟（学会合成，立刻上手）

• 开场：温馨动画，"欢迎来到你的花店"

• 引导拖拽两朵小雏菊合成向日葵，立刻产生视觉反馈

• 再合成一次，得到康乃馨

• 核心目标：让玩家2分钟内理解合成玩法并感到"好玩"

2. 第二阶段：2~5分钟（第一位客人，完成交付）

• 第一位客人（学生少女）上门，头顶显示需要"向日葵"

• 引导拖拽花束交付，金币+花愿到手

• 弹出简短祝贺，展示金币用途

• 核心目标：建立"合成→交付→获得奖励"的基础循环认知

3. 第三阶段：5~10分钟（解锁第一个建筑）

• 金币足够，引导解锁花艺操作台

• 教会点击建筑产花，体验被动产出

• 再接待2~3位客人，熟悉经营节奏

• 核心目标：体验"建筑产花+合成+交付"完整循环

4. 第四阶段：10~20分钟（解锁装修，初见花店变美）

• 花愿足够，引导进入装修页面

• 免费解锁第一件装饰（门面升级），花店外观立刻变化

• 核心目标：感受花店"变美"的成就感，建立长线养成期待

5. 第五阶段：20~30分钟（解锁换装，店主形象焕新）

• 花露足够，引导进入换装页面

• 免费获得第一套服装（清新裙装），店主形象立刻改变

• 引导截图分享

• 核心目标：所有四大玩法全部体验完毕，玩家已形成完整认知和留存动力

6. 设计原则

• 前5分钟决定玩家是否留下，必须在5分钟内体验到完整的"合成→交付→赚钱"循环

• 前30分钟必须让玩家体验到所有核心玩法的"爽感"

• 每个阶段结束都有明确的正反馈（动画、音效、奖励弹窗）

• 新手引导结束后，自然过渡到自由经营模式

十二、开店仪式与日结算（情感体验增强）

1. 每日开店仪式

• 每天首次登录：播放"拉开卷帘门"小动画（约2秒）

• 门面展示当前装修风格，店主穿着当前时装迎客

• 显示"今日天气"（纯装饰，随机晴/阴/雨/雪，影响花店氛围光效）

• 开店后第一位客人立刻到来，给予即时互动感

2. 每日打烊结算

• 每日首次登录时先展示昨日结算（若非首日）：

今日营业额：XXX金币
服务客人：XX位
好评率：XX%（基于交付速度和花等级匹配度）
获得花愿：XX / 获得花露：XX
熟客好感进度提示

• 好评率连续3天90%以上，奖励额外花愿/花露

• 设计说明：低开发成本，但给玩家强烈的"经营一家店"的代入感和成就感

十三、核心数值体系（稳定、不卡关、适配小游戏）

1. 合成时间：即时合成，无等待（拖拽即完成）

2. 建筑产出：点击即时产出，无冷却时间

3. 客人刷新间隔：10~30秒

4. 挂机产出：
• 持续生成基础鲜花（1级花），三个花系随机产出
• 存储上限：根据棋盘格子数计算，满格后停止产出
• 产出速度：基础1朵/60秒，可通过金币提速

5. 离线收益：
• 支持离线产出，上限为2~4小时的产出量
• 超过4小时不再累积，引导玩家每天至少登录2次
• 广告可翻倍已有离线收益（不突破时间上限，只翻倍数量）
• 离线收益领取时播放"开店清点"小动画

• 分享卡片：生成分享图时，使用精致立绘风格的店主形象+花店背景

===== 关键交互细节汇总 =====

1. 拖拽灵敏度：花朵拖拽触发距离 = 10px，避免误触但保证流畅

二十、限定内容产出规则（含限时活动+限定花/时装/装扮）

1. 限定内容范围

包括：限定花束、限定时装、限定花店装修、限定背景、限定店主皮肤、限定摆件。

2. 三套货币完全区分（数值健康、不混乱、不通胀）

1）日常货币一：花愿
• 来源：日常订单、交付客人、任务、成就
• 用途：兑换永久普通装修

2）日常货币二：花露
• 来源：交付客人掉落、熟客订单、任务、成就、好友互访
• 用途：兑换永久普通时装

3）限时活动货币：活动花瓣（节日/版本专属）
• 来源：仅限时活动期间产出
• 用途：兑换全部限定奖励（限定花、限定时装、限定装修）

3. 限定内容唯一来源：限时活动玩法（对标四季物语活动机制）

1. 活动开启后，主棋盘随机产出活动专属花（如：情人节玫瑰、中秋月桂、圣诞雪花花、春日樱花）

2. 活动花可在棋盘内拖拽合成更高阶活动花（活动花独立短链，4级）

3. 活动花可交付给活动专属客人/活动订单

4. 完成后获得：活动花瓣

5. 使用活动花瓣在活动商店兑换：

◦ 限定花束（永久收藏，活动结束后暂绝版）

◦ 限定时装套装

◦ 限定花店装修皮肤

◦ 限定背景/限定特效

4. 活动花与日常花的关系

• 活动花不参与日常合成链

• 活动花不能用于日常订单

• 活动花仅在活动期间存在，活动结束后不再产出

• 已获得的限定花、时装、装修永久保留

5. 限定产出核心规则

• 日常内容：永久可获取、无压力收集

• 限定内容：仅限时活动产出，不进入常规养成

• 活动结束：限定内容暂绝版，后期可节日返场

• 无强制氪金，主要靠活动玩法肝取，支持少量礼包辅助获取

6. 活动设计目的（长期留存、版本迭代、商业化）

• 给玩家阶段性目标，提升长期留存

• 每月/节日更新版本，保持游戏新鲜感

• 限定外观满足收集欲、晒图分享、社交传播

• 支持商业化礼包（限定礼包、快速兑换礼包），不破坏平民体验
